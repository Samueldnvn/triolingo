Chapter 1: Advanced Integration Techniques

📋 Chapter Overview

This chapter expands your integration toolkit beyond the basic methods learned in Calculus I. You'll master sophisticated techniques that can handle a much wider range of integrals, including those involving trigonometric functions, rational expressions, and applications in geometry.

🎯 Learning Objectives

  • Master integration by substitution for complex functions
  • Apply integration by parts using the LIATE rule
  • Solve trigonometric integrals using identities
  • Use trigonometric substitution for expressions like √(a²-x²)
  • Decompose rational functions using partial fractions
  • Evaluate improper integrals with limits
  • Calculate areas between curves using integration
  • Find volumes of revolution using disk/washer and shell methods
  • Compute arc length and surface area of curves

🌍 Real-World Applications

  • Physics: Work done by variable forces, center of mass
  • Engineering: Fluid pressure, moments of inertia
  • Economics: Consumer/producer surplus, accumulated wealth
  • Biology: Population models, drug concentration over time

Lesson 1: Integration by Substitution

📖 Introduction

Integration by substitution is the reverse of the chain rule for differentiation. It's one of the most powerful and frequently used integration techniques. In this lesson, we'll review substitution from Calculus I and explore more advanced applications.

📚 Key Concept: The Substitution Rule

$$\int f(g(x)) \cdot g'(x) \, dx = \int f(u) \, du$$

Steps:

  1. Identify the inner function $u = g(x)$
  2. Compute $du = g'(x) \, dx$
  3. Substitute and rewrite the integral in terms of $u$
  4. Integrate with respect to $u$
  5. Substitute back to express in terms of $x$

🔢 Interactive Example 1: Basic Substitution

Problem: $\int 2x(x^2+1)^3 \, dx$

Solution:

Step 1: Let $u = x^2 + 1$, then $du = 2x \, dx$
Step 2: Substitute: $\int u^3 \, du$
Step 3: Integrate: $\frac{u^4}{4} + C$
Step 4: Substitute back: $\\frac{(x^2+1)^4}{4} + C$

🔢 Interactive Example 2: Trigonometric Substitution

Problem: $\int \sec^2 x \tan x \, dx$

Solution:

Step 1: Let $u = \tan x$, then $du = \sec^2 x \, dx$
Step 2: Substitute: $\int u \, du$
Step 3: Integrate: $\frac{u^2}{2} + C$
Step 4: Substitute back: $\frac{\tan^2 x}{2} + C$

⚠️ Common Mistakes

  • Forgetting to include the differential ($dx$ or $du$) in the substitution
  • Not adjusting for constants that need to be multiplied or divided
  • Forgetting to substitute back to the original variable
  • Choosing $u$ incorrectly - it must be inside $g'(x)$ as well

📎 Additional Resources

  • Practice identifying $u$ in complex expressions
  • Review derivative rules - knowing them backwards helps with substitution
  • Try the "reverse chain rule" perspective: what function when differentiated gives this?

Lesson 2: Integration by Parts

📖 Introduction

What do you do when substitution doesn't work? Enter integration by parts! This technique, derived from the product rule for differentiation, allows us to integrate products of functions that seem impossible at first glance.

📚 Key Concept: Integration by Parts Formula

$$\int u \, dv = uv - \int v \, du$$

🔑 The LIATE Rule for Choosing $u$:

When deciding what to set as $u$, use this priority order:

  • L - Logarithmic functions: $\ln x$, $\log_b x$
  • I - Inverse trigonometric: $\arcsin x$, $\arctan x$, etc.
  • A - Algebraic (polynomials): $x^n$, $3x^2 - 5$, etc.
  • T - Trigonometric: $\sin x$, $\cos x$, $\sec^2 x$, etc.
  • E - Exponential: $e^x$, $2^x$, $e^{3x}$, etc.

The function higher on the list should be $u$; the lower one becomes $dv$.

🔢 Interactive Example 1: $\int x e^x \, dx$

Step 1: Choose $u = x$ (A is higher than E in LIATE)
Step 2: Then $dv = e^x \, dx$, so $du = dx$ and $v = e^x$
Step 3: Apply formula: $x \cdot e^x - \int e^x \, dx$
Step 4: Simplify: $xe^x - e^x + C = e^x(x-1) + C$

🔢 Interactive Example 2: $\int x \sin x \, dx$

Step 1: Choose $u = x$ (A is higher than T in LIATE)
Step 2: Then $dv = \sin x \, dx$, so $du = dx$ and $v = -\cos x$
Step 3: Apply formula: $x(-\cos x) - \int (-\cos x) \, dx$
Step 4: Simplify: $-x\cos x + \sin x + C$

🔢 Interactive Example 3: $\int \ln x \, dx$ (Tricky!)

Notice there's only one function - how do we do parts?
Step 1: Let $u = \ln x$ and $dv = 1 \cdot dx$
Step 2: Then $du = \frac{1}{x} \, dx$ and $v = x$
Step 3: Apply formula: $x \ln x - \int x \cdot \frac{1}{x} \, dx$
Step 4: Simplify: $x \ln x - \int 1 \, dx = x \ln x - x + C$

⚠️ Common Mistakes

  • Misapplying the LIATE rule - remember it's a guideline, not a law
  • Forgetting the minus sign in the formula: $uv - \int v \, du$
  • Incorrectly finding $v$ by integrating $dv$ (don't forget +C, though it cancels)
  • With repeated integration by parts, keeping track of signs can be tricky

Lesson 3: Trigonometric Integrals

📖 Introduction

Integrals involving powers of trigonometric functions appear frequently in calculus. This lesson teaches strategies for evaluating integrals like $\int \sin^2 x \, dx$, $\int \cos^3 x \sin^2 x \, dx$, and similar expressions.

📚 Strategy 1: $\sin^m x \cos^n x$ when $n$ is ODD

If $n$ is odd, save one cosine factor and use $\cos^2 x = 1 - \sin^2 x$

$\int \sin^m x \cos^{2k+1} x \, dx = \int \sin^m x \cos^{2k} x \cos x \, dx = \int \sin^m x (1-\sin^2 x)^k \cos x \, dx$

📚 Strategy 2: $\sin^m x \cos^n x$ when $m$ is ODD

If $m$ is odd, save one sine factor and use $\sin^2 x = 1 - \cos^2 x$

$\int \sin^{2k+1} x \cos^n x \, dx = \int \sin^{2k} x \cos^n x \sin x \, dx = \int (1-\cos^2 x)^k \cos^n x \sin x \, dx$

📚 Strategy 3: Both powers are EVEN

Use the half-angle formulas: $$\sin^2 x = \frac{1 - \cos 2x}{2}, \quad \cos^2 x = \frac{1 + \cos 2x}{2}$$

📚 Strategy 4: $\tan^m x \sec^n x$

  • If $n$ is even, save $\sec^2 x$ and use $\sec^2 x = 1 + \tan^2 x$
  • If $m$ is odd, save $\sec x \tan x$ and use $\tan^2 x = \sec^2 x - 1$

🔢 Example 1: $\int \sin^3 x \, dx$ (odd power)

Step 1: Write as $\int \sin^2 x \cdot \sin x \, dx$
Step 2: Use identity: $\int (1-\cos^2 x) \sin x \, dx$
Step 3: Let $u = \cos x$, $du = -\sin x \, dx$
Step 4: Integral becomes $-\int (1-u^2) \, du = -u + \frac{u^3}{3} + C$
Step 5: Answer: $-\cos x + \frac{\cos^3 x}{3} + C$

🔢 Example 2: $\int \sin^2 x \, dx$ (even power)

Step 1: Use half-angle formula: $\sin^2 x = \frac{1-\cos 2x}{2}$
Step 2: Integral becomes $\int \frac{1-\cos 2x}{2} \, dx$
Step 3: Separate: $\frac{1}{2} \int 1 \, dx - \frac{1}{2} \int \cos 2x \, dx$
Step 4: Integrate: $\frac{x}{2} - \frac{\sin 2x}{4} + C$

🎯 Key Trigonometric Identities to Remember

$\sin^2 x + \cos^2 x = 1$

$1 + \tan^2 x = \sec^2 x$

$1 + \cot^2 x = \csc^2 x$

$\sin 2x = 2\sin x \cos x$

$\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x$

Lesson 4: Trigonometric Substitution

📖 Introduction

Trigonometric substitution is a powerful technique for simplifying integrals containing expressions like $\sqrt{a^2-x^2}$, $\sqrt{a^2+x^2}$, or $\sqrt{x^2-a^2}$. By making clever substitutions, we transform these into simpler trigonometric integrals.

📚 Three Types of Trigonometric Substitution

Type 1: $\sqrt{a^2 - x^2}$

Let $x = a\sin\theta$, where $-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}$
Then $dx = a\cos \theta \, d\theta$ and $\sqrt{a^2-x^2} = a\cos\theta$

Type 2: $\sqrt{a^2 + x^2}$

Let $x = a\tan\theta$, where $-\frac{\pi}{2} < \theta < \frac{\pi}{2}$
Then $dx = a\sec^2 \theta \, d\theta$ and $\sqrt{a^2+x^2} = a\sec\theta$

Type 3: $\sqrt{x^2 - a^2}$

Let $x = a\sec\theta$, where $0 \leq \theta < \frac{\pi}{2}$ or $\pi \leq \theta < \frac{3\pi}{2}$
Then $dx = a\sec\theta \tan\theta \, d\theta$ and $\sqrt{x^2-a^2} = a\tan\theta$

🔢 Example 1: $\int \frac{1}{\sqrt{9-x^2}} \, dx$

Step 1: Identify Type 1 with $a=3$. Let $x = 3\sin\theta$
Step 2: Then $dx = 3\cos\theta \, d\theta$ and $\sqrt{9-x^2} = 3\cos\theta$
Step 3: Substitute: $\int \frac{3\cos\theta}{3\cos\theta} \, d\theta = \int 1 \, d\theta$
Step 4: Integrate: $\theta + C$
Step 5: Since $x = 3\sin\theta$, $\theta = \arcsin(\frac{x}{3})$
Answer: $\arcsin(\frac{x}{3}) + C$

🔢 Example 2: $\int x\sqrt{4+x^2} \, dx$

Step 1: This Type 2 with $a=2$. Let $x = 2\tan\theta$
Step 2: Then $dx = 2\sec^2\theta \, d\theta$ and $\sqrt{4+x^2} = 2\sec\theta$
Step 3: Substitute: $\int (2\tan\theta)(2\sec\theta) \cdot 2\sec^2\theta \, d\theta$
Step 4: Simplify: $8 \int \tan\theta \sec^3\theta \, d\theta$
Step 5: Let $u = \sec\theta$, $du = \sec\theta \tan\theta \, d\theta$
Step 6: Integral becomes $8 \int u^2 \, du = \frac{8u^3}{3} + C$
Step 7: Substitute back: $\frac{8\sec^3\theta}{3} + C$
Step 8: Since $\sec\theta = \frac{\sqrt{4+x^2}}{2}$, answer: $\frac{(4+x^2)^{3/2}}{3} + C$

⚠️ Common Mistakes

  • Choosing the wrong substitution type - look carefully at the form
  • Forgetting to convert $dx$ to the new variable
  • Arithmetic errors when simplifying after substitution
  • Forgetting to convert back to $x$ at the end
  • Using wrong trig identities for back-substitution

Lesson 5: Partial Fractions Decomposition

📖 Introduction

Partial fractions is a technique for breaking down complex rational functions into simpler, more manageable pieces. It's especially useful for integrating rational functions where the denominator can be factored.

📚 The Partial Fractions Method

Step 1: Factor the Denominator

Factor the denominator completely into linear and irreducible quadratic factors.

Step 2: Set Up the Decomposition

For each type of factor:

Linear factor $(x-a)$: $\frac{A}{x-a}$
Repeated linear $(x-a)^n$: $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_n}{(x-a)^n}$
Quadratic factor $(x^2+bx+c)$: $\frac{Ax+B}{x^2+bx+c}$
Repeated quadratic: Similar to linear, with one form for each power

Step 3: Solve for Constants

Multiply both sides by the original denominator and solve for the unknown constants.

🔢 Example 1: $\int \frac{5x-3}{x^2-2x-3} \, dx$

Step 1: Factor: $x^2-2x-3 = (x-3)(x+1)$
Step 2: Set up: $\frac{5x-3}{(x-3)(x+1)} = \frac{A}{x-3} + \frac{B}{x+1}$
Step 3: Multiply: $5x-3 = A(x+1) + B(x-3)$
Step 4: $5x-3 = (A+B)x + (A-3B)$
Step 5: Match coefficients: $A+B=5$ and $A-3B=-3$
Step 6: Solve: Subtract: $4B=8$, so $B=2$, $A=3$
Step 7: Integral = $\int \left(\frac{3}{x-3} + \frac{2}{x+1}\right)dx$
Step 8: Answer: $3\ln|x-3| + 2\ln|x+1| + C$

🌍 Applications of Partial Fractions

  • Solving differential equations
  • Laplace transforms in differential equations and control theory
  • Signal processing and circuit analysis
  • Population dynamics models

Lesson 6: Integration of Rational Functions

📖 Introduction

This lesson brings together all the techniques for integrating rational functions. We'll develop a systematic approach to handle any rational function integrand.

📚 Algorithm for Integrating Rational Functions

1. If degree of numerator ≥ degree of denominator:

Perform long division, then integrate quotient and remainder separately.

2. Factor the denominator completely:

Factor into linear $(ax+b)$ and irreducible quadratic $(ax^2+bx+c)$ factors.

3. Apply partial fractions:

  • Linear factors → use $A/(ax+b)$
  • Repeated linear → use $A_1/(ax+b) + A_2/(ax+b)^2 + ...$
  • Quadratic factors → use $(Ax+B)/(ax^2+bx+c)$

4. Integrate each term:

  • $\int \frac{A}{ax+b} \, dx = \frac{A}{a}\ln|ax+b| + C$
  • $\int \frac{A}{(ax+b)^n} \, dx = \frac{A}{(1-n)a(ax+b)^{n-1}} + C$ (for $n \neq 1$)
  • For quadratics, complete the square and use substitution

🔢 Example: $\int \frac{x^3+1}{x^2-1} \, dx$

Step 1: Long division: $(x^3+1) ÷ (x^2-1) = x + \frac{x+1}{x^2-1}$
Step 2: Factor denominator: $x^2-1 = (x-1)(x+1)$
Step 3: Partial fractions: $\frac{x+1}{(x-1)(x+1)} = \frac{1}{x-1}$
Step 4: Integral = $\int x \, dx + \int \frac{1}{x-1} \, dx$
Step 5: Answer: $\frac{x^2}{2} + \ln|x-1| + C$

Lesson 7: Improper Integrals

📖 Introduction

Improper integrals are integrals with infinite limits or integrands that become infinite somewhere in the interval. They extend the concept of definite integrals to handle infinite domains and unbounded functions.

📚 Type 1: Infinite Intervals

$\int_a^{\infty} f(x) \, dx = \lim_{t \to \infty} \int_a^t f(x) \, dx$

$\int_{-\infty}^b f(x) \, dx = \lim_{t \to -\infty} \int_t^b f(x) \, dx$

$\int_{-\infty}^{\infty} f(x) \, dx = \int_{-\infty}^c f(x) \, dx + \int_c^{\infty} f(x) \, dx$

📚 Type 2: Discontinuous Integrands

If $f$ is discontinuous at $c$ in $[a,b]$:

$\int_a^b f(x) \, dx = \lim_{t \to c^-} \int_a^t f(x) \, dx + \lim_{t \to c^+} \int_t^b f(x) \, dx$

📚 Convergence vs. Divergence

An improper integral converges if the limit exists (is finite). It diverges if the limit is infinite or does not exist.

🔢 Example 1: $\int_1^{\infty} \frac{1}{x^2} \, dx$

Step 1: $\int_1^{\infty} \frac{1}{x^2} \, dx = \lim_{t \to \infty} \int_1^t \frac{1}{x^2} \, dx$
Step 2: Integrate: $\lim_{t \to \infty} \left[-\frac{1}{x}\right]_1^t$
Step 3: Evaluate: $\lim_{t \to \infty} \left(-\frac{1}{t} + 1\right)$
Step 4: As $t \to \infty$, $\frac{1}{t} \to 0$
Step 5: Answer: $0 + 1 = 1$ (converges)

🔢 Example 2: $\int_0^1 \frac{1}{\sqrt{x}} \, dx$ (Type 2)

Step 1: Discontinuous at $x=0$. $\int_0^1 \frac{1}{\sqrt{x}} \, dx = \lim_{t \to 0^+} \int_t^1 x^{-1/2} \, dx$
Step 2: Integrate: $\lim_{t \to 0^+} [2\sqrt{x}]_t^1$
Step 3: Evaluate: $\lim_{t \to 0^+} (2 - 2\sqrt{t})$
Step 4: As $t \to 0^+$, $\sqrt{t} \to 0$
Step 5: Answer: $2$ (converges)

🎯 Important Test for Convergence (p-test)

$\int_1^{\infty} \frac{1}{x^p} \, dx$ converges if $p > 1$, diverges if $p \leq 1$

$\int_0^1 \frac{1}{x^p} \, dx$ converges if $p < 1$, diverges if $p \geq 1$

Lesson 8: Applications - Area Between Curves

📖 Introduction

One of the most beautiful applications of integration is finding the area between two curves. This technique has applications in physics, economics, engineering, and many other fields.

📚 Area Between Curves

$$A = \int_a^b [f(x) - g(x)] \, dx$$

Where $f(x)$ is the upper function and $g(x)$ is the lower function on $[a,b]$.

Key Steps:

  1. Find intersection points to determine limits of integration
  2. Determine which function is above the other
  3. Set up and evaluate the integral of (upper - lower)
  4. If curves cross, separate into multiple integrals

When using $y$ as the independent variable:

$$A = \int_c^d [h(y) - k(y)] \, dy$$

🔢 Example: Find the area between $y = x^2$ and $y = x$

Step 1: Find intersections: $x^2 = x$, so $x^2 - x = 0$, $x(x-1) = 0$
Step 2: Intersection points: $x=0$ and $x=1$
Step 3: On $[0,1]$, the line $y=x$ is above the parabola $y=x^2$
Step 4: Area = $\int_0^1 (x - x^2) \, dx$
Step 5: Integrate: $[\frac{x^2}{2} - \frac{x^3}{3}]_0^1$
Step 6: Evaluate: $(\frac{1}{2} - \frac{1}{3}) - (0 - 0) = \frac{1}{6}$

🌍 Applications

  • Physics: Work done, center of mass
  • Economics: Consumer and producer surplus
  • Probability: Area under probability density functions
  • Engineering: Stress distributions, fluid dynamics

Lesson 9: Applications - Volumes of Revolution (Disks/Washers)

📖 Introduction

When we rotate a region around an axis, we create a 3D solid. The disk and washer methods allow us to calculate the volume of these solids using integration.

📚 Disk Method (Rotation about x-axis)

$$V = \pi \int_a^b [f(x)]^2 \, dx$$

When rotating $y = f(x)$ around the x-axis (or $x = g(y)$ around y-axis).

📚 Washer Method

$$V = \pi \int_a^b \left([f(x)]^2 - [g(x)]^2\right) \, dx$$

When there's a hole in the middle - rotation of region between two curves.

Remember: Outer radius squared minus inner radius squared!

🔢 Example 1: Volume of solid formed by rotating $y = \sqrt{x}$ from $x=0$ to $x=4$ about x-axis

Step 1: Use disk method: $V = \pi \int_0^4 (\sqrt{x})^2 \, dx$
Step 2: Simplify: $V = \pi \int_0^4 x \, dx$
Step 3: Integrate: $\pi [\frac{x^2}{2}]_0^4$
Step 4: Evaluate: $\pi (\frac{16}{2} - 0) = 8\pi$

Lesson 10: Applications - Volumes (Cylindrical Shells)

📖 Introduction

The shell method provides an alternative way to find volumes of revolution. Sometimes it's easier than the disk/washer method, especially when the axis of rotation is not parallel to the coordinate axes.

📚 Shell Method (Rotation about y-axis)

$$V = 2\pi \int_a^b x \cdot f(x) \, dx$$

Where $x$ is the radius and $f(x)$ is the height of each cylindrical shell.

General Formula:

$V = 2\pi \int_a^b (\text{radius}) \cdot (\text{height}) \, dx$

🔢 Example: Use shell method to find volume of region under $y = x^3$ from $x=0$ to $x=1$, rotated about y-axis

Step 1: Shell method: $V = 2\pi \int_0^1 x \cdot x^3 \, dx$
Step 2: Simplify: $V = 2\pi \int_0^1 x^4 \, dx$
Step 3: Integrate: $2\pi [\frac{x^5}{5}]_0^1$
Step 4: Evaluate: $2\pi (\frac{1}{5} - 0) = \frac{2\pi}{5}$

🎯 When to Use Which Method?

  • Disk/Washer: When the radius of washers is expressed as a function of the integration variable
  • Shell: When the height of shells is easier to express

Lesson 11: Applications - Arc Length

📖 Introduction

Calculating the length of a curve is another important application of integration. This has uses in physics (path length), engineering (cable length), and geometry.

📚 Arc Length Formula

$$L = \int_a^b \sqrt{1 + \left(\frac{dy}{dx}\right)^2} \, dx$$

For a curve $y = f(x)$ from $x=a$ to $x=b$.

Alternative form (integral with respect to y):

$$L = \int_c^d \sqrt{1 + \left(\frac{dx}{dy}\right)^2} \, dy$$

🔢 Example: Find the length of $y = \frac{2}{3}x^{3/2}$ from $x=0$ to $x=3$

Step 1: Find derivative: $\frac{dy}{dx} = x^{1/2}$
Step 2: Square it: $\left(\frac{dy}{dx}\right)^2 = x$
Step 3: Set up integral: $L = \int_0^3 \sqrt{1+x} \, dx$
Step 4: Let $u = 1+x$, $du = dx$: $L = \int_1^4 u^{1/2} \, du$
Step 5: Integrate: $[\frac{2}{3}u^{3/2}]_1^4$
Step 6: Evaluate: $\frac{2}{3}(8 - 1) = \frac{14}{3}$

Lesson 12: Applications - Surface Area

📖 Introduction

When a curve is rotated about an axis, it creates a surface. The surface area of this surface of revolution can be calculated using integration.

📚 Surface Area Formulas

About the x-axis:

$$S = 2\pi \int_a^b f(x) \sqrt{1 + \left[f'(x)\right]^2} \, dx$$

About the y-axis:

$$S = 2\pi \int_a^b x \sqrt{1 + \left[f'(x)\right]^2} \, dx$$

Mnemonic: $S = 2\pi \int (\text{radius}) \cdot (\text{arc length element}) \, dx$

🔢 Example: Surface area of $y = x^3$ from $x=0$ to $x=1$, rotated about x-axis

Step 1: $f(x) = x^3$, $f'(x) = 3x^2$
Step 2: $S = 2\pi \int_0^1 x^3 \sqrt{1 + 9x^4} \, dx$
Step 3: Let $u = 1+9x^4$, $du = 36x^3 dx$
Step 4: $S = 2\pi \int_1^{10} u^{1/2} \cdot \frac{du}{36}$
Step 5: $S = \frac{\pi}{18} [\frac{2}{3}u^{3/2}]_1^{10}$
Step 6: $S = \frac{\pi}{27}(10\sqrt{10} - 1)$
