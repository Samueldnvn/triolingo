Chapter 3: Series Convergence Tests

📋 Chapter Overview

This chapter provides a toolkit of tests to determine whether an infinite series converges or diverges. Each test works best for certain types of series, and part of mastering this chapter is learning which test to apply in which situation.

🎯 Learning Objectives

  • Apply the Divergence Test (and understand its limitations)
  • Use the Integral Test for positive, decreasing functions
  • Compare series using Direct and Limit Comparison Tests
  • Test alternating series for convergence
  • Distinguish between absolute and conditional convergence
  • Apply the Ratio Test for series with factorials and exponentials
  • Use the Root Test for series with powers of n
  • Develop intuition for choosing the right test

🌍 Why This Matters

  • Power Series: Determining radius and interval of convergence
  • Approximation: Taylor series rely on convergence
  • Differential Equations: Series solutions
  • Signal Processing: Fourier series

Lesson 17: The Divergence Test

📖 Introduction

The Divergence Test (also called the nth-Term Test) is the simplest test we have. It can only tell us if a series diverges - it can never prove convergence!

📚 The Divergence Test

THEOREM: If $\lim_{n \to \infty} a_n \neq 0$ or does not exist, then $\sum_{n=1}^{\infty} a_n$ diverges.

⚠️ CRITICAL WARNING

THE CONVERSE IS FALSE!

If $\lim_{n \to \infty} a_n = 0$, the series MAY converge or MAY diverge.

Example: $\sum_{n=1}^{\infty} \frac{1}{n}$ (harmonic series) has $\lim_{n \to \infty} \frac{1}{n} = 0$, but diverges!

Summary of What the Test Does:

  • ✅ Can prove: DIVERGENCE (if $\lim a_n \neq 0$)
  • ❌ Cannot prove: CONVERGENCE (even if $\lim a_n = 0$)

🔢 Example 1: Test $\sum_{n=1}^{\infty} \frac{n+1}{2n+1}$ for convergence

Step 1: Apply Divergence Test: Find $\\lim_{n \\to \\infty} \\frac{n+1}{2n+1}$
Step 2: Divide by n: $\\lim_{n \\to \\infty} \\frac{1 + 1/n}{2 + 1/n} = \\frac{1}{2}$
Step 3: Since limit = $\\frac{1}{2} \\neq 0$, the series DIVERGES

Lesson 18: The Integral Test

📖 Introduction

The Integral Test connects series and integrals! If we have a positive, decreasing function, the convergence of the series $\sum a_n$ is equivalent to the convergence of the integral $\int f(x) dx$.

📚 The Integral Test

THEOREM: Let $f$ be a continuous, positive, decreasing function on $[1, \infty)$ and let $a_n = f(n)$.

Then $\sum_{n=1}^{\infty} a_n$ converges $\iff$ $\int_1^{\infty} f(x) \, dx$ converges.

Requirements:

  1. $f(x)$ is positive for $x \geq 1$
  2. $f(x)$ is continuous for $x \geq 1$
  3. $f(x)$ is decreasing for $x \geq 1$
  4. $a_n = f(n)$ (discrete terms match the function)

p-Series Test (Special Case):

$$\sum_{n=1}^{\infty} \frac{1}{n^p}$$

Converges if $p > 1$, diverges if $p \leq 1$

🔢 Example: Test $\sum_{n=1}^{\infty} \frac{1}{n^2}$ for convergence

Step 1: Let $f(x) = \\frac{1}{x^2}$, which is positive, continuous, and decreasing on $[1, \\infty)$
Step 2: Apply Integral Test: $\\int_1^{\\infty} \\frac{1}{x^2} \\, dx = \\lim_{t \\to \\infty} \\int_1^t x^{-2} \\, dx$
Step 3: Integrate: $\\lim_{t \\to \\infty} [-\\frac{1}{x}]_1^t = \\lim_{t \\to \\infty} (-\\frac{1}{t} + 1) = 1$
Step 4: The integral converges (to 1), so the series CONVERGES

Lesson 19: Comparison Tests

📖 Introduction

Comparison tests allow us to determine convergence by comparing to series we already know. They're powerful when we can relate a complex series to a simpler, known series.

📚 Direct Comparison Test

Let $\sum a_n$ and $\sum b_n$ be series with positive terms.

If $0 < a_n \leq b_n$ for all $n$:
- If $\sum b_n$ converges, then $\sum a_n$ converges
- If $\sum a_n$ diverges, then $\sum b_n$ diverges

Key Idea:

  • Smaller convergent → smaller convergent (easy case)
  • Larger divergent → larger divergent (easy case)

📚 Limit Comparison Test

Let $\sum a_n$ and $\sum b_n$ be series with positive terms.

If $\\lim_{n \\to \\infty} \\frac{a_n}{b_n} = c$ where $0 < c < \\infty$:

Then $\sum a_n$ and $\sum b_n$ either both converge or both diverge.

Benefits of Limit Comparison:

  • Don't need one series to be always larger or smaller
  • Works with limit analysis, which we're good at!

🔢 Example: Test $\sum_{n=1}^{\infty} \frac{1}{n^2 + 1}$ using Limit Comparison

Step 1: Compare to $b_n = \\frac{1}{n^2}$ (p-series with p=2 > 1, converges)
Step 2: Compute limit: $\\lim_{n \\to \\infty} \\frac{1/(n^2+1)}{1/n^2} = \\lim_{n \\to \\infty} \\frac{n^2}{n^2+1}$
Step 3: Simplify: $\\lim_{n \\to \\infty} \\frac{1}{1 + 1/n^2} = 1$
Step 4: Since limit = 1 (positive and finite), both series behave the same
Step 5: Since $\sum \\frac{1}{n^2}$ converges, $\sum \\frac{1}{n^2+1}$ also CONVERGES

Lesson 20: Alternating Series Test

📖 Introduction

An alternating series is one where signs alternate between positive and negative. The Alternating Series Test (or Leibniz Test) gives a simple criterion for convergence.

📚 Alternating Series Form

$$\sum_{n=1}^{\infty} (-1)^{n-1} b_n = b_1 - b_2 + b_3 - b_4 + \cdots$$
or
$$\sum_{n=1}^{\infty} (-1)^n b_n = -b_1 + b_2 - b_3 + b_4 - \cdots$$
where $b_n > 0$

📚 Alternating Series Test (Leibniz Test)

THEOREM: If the alternating series $\sum (-1)^{n-1} b_n$ satisfies:

1. $b_{n+1} \leq b_n$ for all $n$ (decreasing)
2. $\lim_{n \to \infty} b_n = 0$

then the series converges.

Common Alternating Series:

  • Alternating Harmonic: $\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n}$ (converges!)
  • Alternating p-series: $\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n^p}$ (converges for all $p > 0$)

Lesson 21: Absolute and Conditional Convergence

📖 Introduction

For series with both positive and negative terms, we have two types of convergence. Understanding the distinction is crucial for calculus applications.

📚 Definitions

Absolute Convergence:

$\sum a_n$ converges absolutely if $\sum |a_n|$ converges.

Conditional Convergence:

$\sum a_n$ converges conditionally if $\sum a_n$ converges but $\sum |a_n|$ diverges.

📚 Important Theorem

THEOREM: If $\sum |a_n|$ converges, then $\sum a_n$ also converges.

In other words: Absolute convergence implies convergence.

Relationship:

  • Absolutely convergent → Convergent (always)
  • Convergent → May be absolute or conditional

🔢 Example: Analyze $\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n}$ (Alternating Harmonic Series)

Step 1: Does it converge? Yes, by Alternating Series Test
Step 2: Check absolute convergence: $\sum |\\frac{(-1)^{n-1}}{n}| = \sum \\frac{1}{n}$
Step 3: This is the harmonic series, which DIVERGES (p=1)
Step 4: Conclusion: The series converges CONDITIONALLY (not absolutely)

Lesson 22: The Ratio Test

📖 Introduction

The Ratio Test is powerful for series involving factorials, exponentials, and powers. It works by examining how successive terms compare.

📚 The Ratio Test

Let $L = \lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right|$.

Then:
- If $L < 1$: Series converges absolutely
- If $L > 1$ or $L = \infty$: Series diverges
- If $L = 1$: Test is inconclusive

When to Use It:

  • Series with factorials: $n!$, $(2n)!$, etc.
  • Series with exponentials: $2^n$, $3^n$, $e^n$, etc.
  • Series where $a_{n+1}$ is related to $a_n$

🔢 Example: Test $\sum_{n=1}^{\infty} \frac{2^n}{n!}$ for convergence

Step 1: Apply Ratio Test: $L = \\lim_{n \\to \\infty} \\left|\\frac{a_{n+1}}{a_n}\\right| = \\lim_{n \\to \\infty} \\frac{2^{n+1}/(n+1)!}{2^n/n!}$
Step 2: Simplify: $\\lim_{n \\to \\infty} \\frac{2^{n+1} \\cdot n!}{2^n \\cdot (n+1)!} = \\lim_{n \\to \\infty} \\frac{2}{n+1}$
Step 3: Compute limit: $\\lim_{n \\to \\infty} \\frac{2}{n+1} = 0$
Step 4: Since $L = 0 < 1$, the series CONVERGES ABSOLUTELY

Lesson 23: The Root Test

📖 Introduction

The Root Test is similar to the Ratio Test but uses nth roots instead. It's especially useful for series where terms are raised to the power of n.

📚 The Root Test

Let $L = \lim_{n \to \infty} \sqrt[n]{|a_n|}$.

Then:
- If $L < 1$: Series converges absolutely
- If $L > 1$ or $L = \infty$: Series diverges
- If $L = 1$: Test is inconclusive

When to Use It:

  • Series of the form $\sum (f(n))^n$
  • Series where terms have n as an exponent
  • When simplifying nth roots is easier than ratio

🔢 Example: Test $\sum_{n=1}^{\infty} \left(\frac{n}{2n+1}\right)^n$ for convergence

Step 1: Apply Root Test: $L = \\lim_{n \\to \\infty} \\sqrt[n]{\\left(\\frac{n}{2n+1}\\right)^n}$
Step 2: Simplify nth root: $\\lim_{n \\to \\infty} \\frac{n}{2n+1}$
Step 3: Divide by n: $\\lim_{n \\to \\infty} \\frac{1}{2 + 1/n} = \\frac{1}{2}$
Step 4: Since $L = \\frac{1}{2} < 1$, the series CONVERGES

Lesson 24: Strategy Guide - Choosing the Right Test

📖 Introduction

With so many convergence tests available, knowing which one to apply is a skill. This lesson provides a systematic approach to choosing the right test.

📚 Decision Tree

Step 1: Check the Divergence Test FIRST

If $\lim_{n \\to \\infty} a_n \\neq 0$, the series DIVERGES. Done!
If $\lim a_n = 0$, we need more tests.

Step 2: Identify the type of series

  • Alternating series: Use Alternating Series Test
  • Positive terms: Continue to Step 3

Step 3: Check for special forms

  • p-series: $\sum \frac{1}{n^p}$ - use p-test
  • Geometric: $\sum ar^n$ - use geometric series test
  • Factorials/exponentials: Try Ratio Test
  • n as exponent: Try Root Test

Step 4: If none of the above, try Comparison Tests

  • Direct comparison: If you can establish inequality relationship
  • Limit comparison: More flexible, use limit analysis

Step 5: Last resort

  • Integral Test: If function is positive, continuous, decreasing

🎯 Quick Reference

Series Type Best Test Example
$\lim a_n \neq 0$ Divergence Test $\sum n/(n+1)$
Alternating Alternating Series Test $\sum (-1)^n/n$
$\sum 1/n^p$ p-Test $\sum 1/n^2$
Factorials, $2^n$, etc. Ratio Test $\sum 2^n/n!$
$(f(n))^n$ Root Test $\sum (n/2n)^n$
Rational function Limit Comparison $\sum (n+1)/n^3$
Nice integral exists Integral Test $\sum 1/(n^2+1)$

📓 Chapter 3 Summary

  • Divergence Test: If $\lim a_n \neq 0$, diverges (but converse is false!)
  • Integral Test: Connects series to integrals (for positive, continuous, decreasing)
  • Comparison Tests: Compare to known series (direct and limit comparison)
  • Alternating Series: Converges if decreasing and terms → 0
  • Absolute vs Conditional: Absolute convergence requires $\sum |a_n|$ to converge
  • Ratio Test: Best for factorials and exponentials
  • Root Test: Best for $(f(n))^n$ forms
  • Strategy: Use a systematic approach - always check Divergence Test first!
