IoT Hardware Fundamentals

Understanding Sensors and Devices

Sensors are devices that detect and measure physical properties, converting them into electrical signals for processing by IoT systems.

TypeDigitalAnalog
OutputDiscrete 0/1 or multiple valuesContinuous voltage level
InterfaceDigital I/O, SPI, I2CADC required
Noise ImmunityHighLower, susceptible to noise
COMMON SENSOR CATEGORIES
  • Environmental: Temperature (DS18B20, DHT22), Humidity (DHT11), Pressure (BMP280)
  • Motion: Accelerometer (MPU6050), Gyroscope, PIR Motion (HC-SR501), Ultrasonic (HC-SR04)
  • Light: Photoresistor (LDR), Photodiode, RGB Color (TCS3200), Light Intensity (BH1750)
  • Other: Gas (MQ-135), Sound, Distance, Touch

SENSOR PROGRAMMING

Java can interface with sensors through GPIO and communication protocols for reading data and controlling actuators.

// Digital sensor reading with Pi4J
import com.pi4j.io.gpio.*;
public class DigitalSensorReader{
  private final GpioPinDigitalInput pin;
  public DigitalSensorReader(){
    GpioController gpio=GpioFactory.getInstance();
    pin=gpio.provisionDigitalInputPin(RaspiPin.GPIO_04);
  }
  public boolean readState(){return pin.getState().isHigh();}
}

ANALOG-TO-DIGITAL CONVERSION

ADC converts continuous analog signals to discrete digital values. Essential for reading analog sensors like thermistors.

ADC Digital Value
$$DigitalValue = \frac{Vin}{Vref} \times (2^n - 1)$$
Where Vin=Input voltage, Vref=Reference voltage, n=Bits
ADC CHARACTERISTICS
  • Resolution: 8-bit=256, 10-bit=1024, 12-bit=4096 values
  • Reference Voltage: Maximum measurable voltage
  • Sampling Rate: Samples per second
// Reading analog sensor (MCP3008 via SPI)
public class AnalogSensorReader{
  public double readAnalog(int channel){
    int rawValue=readMCP3008(channel);
    double voltage=(rawValue/1023.0)*3.3;
    return voltage;
  }
}
