BLUETOOTH LOW ENERGY (BLE)

BLE is designed for short-range communication with very low power consumption, ideal for battery-powered IoT devices.

BLE CHARACTERISTICS
  • Power: Consumes microamps to milliamps (years of battery life)
  • Range: Up to 100m (class 1)
  • Topology: Point-to-point, star, mesh
  • Data Rate: Up to 2 Mbps
  • Use Cases: Wearables, smart home, asset tracking
// Java BLE with TinyB library
import com.mbientlab.ble.*;
BluetoothManager ble=BluetoothManager.getInstance();
ble.startScan(device->{
  if(device.getName().contains("Sensor")){
    device.connect();
    // Read sensor data via GATT
  }
});

LOW POWER WIDE AREA NETWORKS (LPWAN)

LPWAN technologies provide long-range, low-power connectivity for IoT applications like smart cities and agriculture.

ZIGBEE
  • IEEE 802.15.4 standard
  • 2.4 GHz, 868/915 MHz
  • Mesh networking for extended range
  • Smart home, industrial automation
LORAWAN
  • Long range up to 15km rural
  • Ultra low power (10+ years battery)
  • Star topology with gateways
  • Smart cities, agriculture, tracking
SIGFOX
  • Global cellular network
  • Very low bandwidth (up to 12 bytes)
  • Ultra low power consumption
  • Asset tracking, alarms
