What Makes Calculus III Different?

The key difference between single-variable calculus (Calculus I & II) and multivariable calculus (Calculus III) is the number of dimensions we work in. Here's what changes:

Concept Single-Variable (1D) Multivariable (2D/3D)
Function y = f(x) z = f(x,y)
Graph Curve in 2D plane Surface in 3D space
Derivative Slope of tangent line Slope of tangent plane (multiple directions!)
Integral Area under curve Volume under surface
Limit Approach from left/right Approach from ANY direction in space!

Calculus Review: Essential Foundation

Mastering the Tools You'll Need

Derivatives: The Foundation of Change

Definition: The Derivative

The derivative measures the instantaneous rate of change of a function at a point. Geometrically, it's the slope of the tangent line to the curve.

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

Power Rule (Essential!)

$$\frac{d}{dx}[x^n] = nx^{n-1}$$

Example: $$\frac{d}{dx}[x^3] = 3x^2$$ and $$\frac{d}{dx}[x^{-2}] = -2x^{-3} = -\frac{2}{x^3}$$

Chain Rule (Critical for Calc III!)

$$\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$$

Example: $$\frac{d}{dx}[\sin(x^2)] = \cos(x^2) \cdot 2x = 2x\cos(x^2)$$

This rule will be EVERYWHERE in partial derivatives!

Integrals: Accumulation and Area

Definition: The Definite Integral

The definite integral represents the signed area under a curve between two points.

$$\int_a^b f(x) dx = \lim_{n \to \infty} \sum_{i=1}^n f(x_i^*) \Delta x$$

Integration by Parts

$$\int u dv = uv - \int v du$$

Example: $$\int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C = (x-1)e^x + C$$

The Fundamental Theorem of Calculus

Bridging Derivatives and Integrals

This theorem connects the two main concepts of calculus and is absolutely crucial. It states:

$$\int_a^b f(x) dx = F(b) - F(a)$$

where F is any antiderivative of f (meaning F' = f).

Limits in Multiple Dimensions

The Fundamental Difference

🔨 NEW TERRITORY AHEAD 🔨

Limits become MUCH more interesting (and challenging) in multiple dimensions. In single-variable calculus, a limit exists if the function approaches the same value from BOTH the left AND the right.

🚨 IN 2D/3D: The limit must exist from EVERY possible direction!

The 2D Limit Definition

Definition: Multivariable Limit

$$\lim_{(x,y) \to (a,b)} f(x,y) = L$$

This means that for ANY path approaching (a,b), the function values approach L. This includes straight lines, curves, approaching from above, below, diagonally - EVERYTHING!

Example: Simple Limit

Find $$\lim_{(x,y) \to (1,2)} (3x + 4y)$$

Solution: Since this is a polynomial (continuous everywhere), we just substitute:

$$3(1) + 4(2) = 3 + 8 = 11$$

The limit is 11 from ANY direction!

Example: Path-Dependent Limit

Find $$\lim_{(x,y) \to (0,0)} \frac{xy}{x^2 + y^2}$$

Path 1: Along y = 0 (x-axis)

$$\frac{x(0)}{x^2 + 0} = \frac{0}{x^2} = 0$$

Path 2: Along y = x (diagonal)

$$\frac{x(x)}{x^2 + x^2} = \frac{x^2}{2x^2} = \frac{1}{2}$$
❌ LIMIT DOES NOT EXIST - Different paths give different values!

Continuity in 2D

Definition: Continuity at (a,b)

$$\lim_{(x,y) \to (a,b)} f(x,y) = f(a,b)$$

A function is continuous at a point if: (1) the limit exists, (2) the function is defined there, and (3) they're equal.

📋 KEY TAKEAWAYS

  • Polynomials, sin, cos, exp are continuous EVERYWHERE
  • Rational functions are continuous WHEREVER the denominator ≠ 0
  • Composition of continuous functions is continuous
  • ALWAYS check path-dependency for suspicious limits!
