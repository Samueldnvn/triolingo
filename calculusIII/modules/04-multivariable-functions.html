Module 4: Functions of Several Variables

Surfaces, Contours, Limits & Continuity | Calculus III

Overview
Beyond Single Variables

Functions of multiple variables describe surfaces in 3D space. Understanding their behavior requires new tools for visualization and analysis.

Functions of Two Variables
Definition
Multivariable Function
Notation
$$z = f(x,y)$$ or $$f: \mathbb{R}^2 \to \mathbb{R}$$
Example
Common Functions
Paraboloid

$$z = x^2 + y^2$$ (bowl-shaped)

Hyperbolic Paraboloid

$$z = x^2 - y^2$$ (saddle)

Plane

$$z = ax + by + c$$ (flat surface)

Sphere

$$x^2 + y^2 + z^2 = r^2$$

Level Curves and Contours
Definition
Level Curves

For constant k, the set $$f(x,y) = k$$ called a contour or level curve

Example
For $$z = x^2 + y^2$$, level curves are circles: $$x^2 + y^2 = k$$
Visualization

A topographic map is a set of level curves showing elevation. Contours help visualize surfaces in 2D.

Limits and Continuity
Definition
Limit in 2D
Definition
$$\lim_{(x,y)\to(a,b)} f(x,y) = L$$ if for all paths to (a,b), f approaches L
Important
Path Dependence

In 2D, limits must exist from ALL directions. Different paths can give different values, meaning the limit doesn't exist!

Example
Path-Dependent Limit

$$\displaystyle\lim_{(x,y)\to(0,0)} \frac{xy}{x^2+y^2}$$

Along $$y=0$$: limit is 0
Along $$y=x$$: limit is 1/2

≠ Different paths → limit DNE

Continuity
Condition
$$\lim_{(x,y)\to(a,b)} f(x,y) = f(a,b)$$ for f to be continuous at (a,b)
Practice
Problem 1
