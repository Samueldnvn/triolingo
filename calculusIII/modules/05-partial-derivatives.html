Module 5: Partial Derivatives & Gradients

Rates of Change in Multiple Directions | Calculus III

Overview
Differentiating Multivariable Functions

Partial derivatives measure how a function changes when varying one variable while keeping others constant. The gradient vector combines all partial derivatives.

Partial Derivatives
Definition
First Partial Derivatives
With respect to x
$$f_x = \frac{\partial f}{\partial x} = \lim_{h\to0} \frac{f(x+h,y) - f(x,y)}{h}$$
With respect to y
$$f_y = \frac{\partial f}{\partial y} = \lim_{h\to0} \frac{f(x,y+h) - f(x,y)}{h}$$
Interpretation

$$f_x$$ = slope of tangent in x-direction (holding y constant)
$$f_y$$ = slope of tangent in y-direction (holding x constant)

Example
Computing Partial Derivatives
Problem

$$f(x,y) = x^2y + xy^3$$

Solution

$$f_x = 2xy + y^3$$ (treat y as constant)
$$f_y = x^2 + 3xy^2$$ (treat x as constant)

Higher-Order Partial Derivatives
Definition
Second Partial Derivatives
Pure Second
$$f_{xx} = \frac{\partial^2 f}{\partial x^2}, \quad f_{yy} = \frac{\partial^2 f}{\partial y^2}$$
Mixed Second
$$f_{xy} = \frac{\partial^2 f}{\partial x \partial y}, \quad f_{yx} = \frac{\partial^2 f}{\partial y \partial x}$$
Theorem
Clairaut's Theorem (Symmetry)
Condition
If $$f_{xy}$$ and $$f_{yx}$$ are continuous, then $$f_{xy} = f_{yx}$$
Example
Second Derivatives

$$f(x,y) = x^3 + 2xy^2$$

$$f_x = 3x^2 + 2y^2, \quad f_{xx} = 6x, \quad f_{xy} = 4y$$

$$f_y = 4xy, \quad f_{yy} = 4x, \quad f_{yx} = 4y$$

✓ $$f_{xy} = f_{yx} = 4y$$ (satisfied)

The Gradient Vector
Definition
Gradient
Formula
$$\nabla f = \langle f_x, f_y \rangle$$ (2D) or $$\nabla f = \langle f_x, f_y, f_z \rangle$$ (3D)
Geometric Meaning
  • Points in direction of steepest increase
  • Magnitude = rate of steepest increase
  • Perpendicular to level curves/surfaces
Directional Derivatives
Definition
Directional Derivative Along Unit Vector $$\vec{u} = \langle a,b \rangle$$
Formula
$$D_{\vec{u}}f = \nabla f \cdot \vec{u} = f_x a + f_y b$$
Maximum Rate of Change
Maximum at direction of $$\nabla f$$, with magnitude $$|\nabla f|$$
Example
Directional Derivative

$$f(x,y) = x^2 + y^2, \text{ at } (1,1), \text{ along } \vec{u} = \langle 1,0 \rangle$$

$$\nabla f = \langle 2x, 2y \rangle = \langle 2,2 \rangle$$

$$D_{\vec{u}}f = \langle 2,2 \rangle \cdot \langle 1,0 \rangle = 2(1) + 2(0) = 2$$
