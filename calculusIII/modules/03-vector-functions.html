Vector-Valued Functions

Functions That Output Vectors

🔨 NEW FUNCTION TYPE 🔨

Vector functions input a scalar and output a vector. They describe curves in space!

Definition: Vector Function

$$\vec{r}(t) = \langle f(t), g(t), h(t) \rangle = f(t)\vec{i} + g(t)\vec{j} + h(t)\vec{k}$$

Each component is a regular scalar function of t

Example: Helix

$$\vec{r}(t) = \langle \cos t, \sin t, t \rangle$$

As t increases, this traces a spiral (helix) going up the z-axis

Domain of Vector Functions

Definition: Domain

The domain is the intersection of domains of all component functions

Example

$$\vec{r}(t) = \langle \sqrt{t}, \frac{1}{t-2}, \ln(t+1) \rangle$$

Domain: t ≥ 0 AND t ≠ 2 AND t > -1 → t ≥ 0, t ≠ 2

Derivatives & Integrals

Velocity, Acceleration, and Displacement

Derivative of Vector Function

Definition: Vector Derivative

$$\vec{r}'(t) = \frac{d\vec{r}}{dt} = \langle f'(t), g'(t), h'(t) \rangle$$
Physical Interpretation
  • $$\vec{r}(t)$$ = Position vector
  • $$\vec{v}(t) = \vec{r}'(t)$$ = Velocity vector (tangent to path)
  • $$\vec{a}(t) = \vec{v}'(t) = \vec{r}''(t)$$ = Acceleration vector

Example: Projectile Motion

Position: $$\vec{r}(t) = \langle v_{0x}t, v_{0y}t - \tfrac{1}{2}gt^2 \rangle$$

Velocity: $$\vec{v}(t) = \langle v_{0x}, v_{0y} - gt \rangle$$

Acceleration: $$\vec{a}(t) = \langle 0, -g \rangle$$ (constant downward!)

Integral of Vector Function

Definition: Vector Integral

$$\int \vec{r}(t) dt = \langle \int f(t) dt, \int g(t) dt, \int h(t) dt \rangle + \vec{C}$$

Example

$$\int \langle 2t, 3t^2, e^t \rangle dt = \langle t^2+C_1, t^3+C_2, e^t+C_3 \rangle$$

🛠️ EACH COMPONENT GETS ITS OWN CONSTANT OF INTEGRATION!

Arc Length

Measuring the Length of Curves

Definition: Arc Length

$$L = \int_a^b |\vec{r}'(t)| dt$$

where $$|\vec{r}'(t)| = \sqrt{(f')^2 + (g')^2 + (h')^2}$$ is speed

Example: Line Segment

Position: $$\vec{r}(t) = \langle t, 2t, 3t \rangle$$ for 0 ≤ t ≤ 1

Velocity: $$\vec{r}'(t) = \langle 1, 2, 3 \rangle$$

Speed: $$|\vec{r}'(t)| = \sqrt{1+4+9} = \sqrt{14}$$ (constant!)

$$L = \int_0^1 \sqrt{14} dt = \sqrt{14}$$

📋 ARC LENGTH FORMULA APPLIES TO:

  • 2D curves in the plane
  • 3D curves in space
  • Any parametrized curve!
  • For 2D function y = f(x): $$L = \int_a^b \sqrt{1 + [f'(x)]^2} dx$$

Curvature

Measuring How Fast a Curve Turns

🔨 ADVANCED CONCEPT 🔨

Definition: Curvature κ

$$\kappa = \frac{|\vec{T}'(t)|}{|\vec{r}'(t)|} = \frac{|\vec{r}'(t) \times \vec{r}''(t)|}{|\vec{r}'(t)|^3}$$

Units: inverse length (1/m)

Unit Tangent Vector

$$\vec{T}(t) = \frac{\vec{r}'(t)}{|\vec{r}'(t)|}$$

Points in direction of motion, has magnitude 1

Example: Circle of radius R

Position: $$\vec{r}(t) = \langle R\cos t, R\sin t, 0 \rangle$$

$$\kappa = \frac{1}{R}$$

Smaller radius = tighter curve = higher curvature!

🛠️ LINE HAS ZERO CURVATURE: κ = 0 STRAIGHT LINES NEVER BEND!
