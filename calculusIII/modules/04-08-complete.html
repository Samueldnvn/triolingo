Module 4: Functions of Several Variables

Surfaces, Contours, and 2D Limits

Definition: z = f(x,y)

$$z = f(x,y) = x^2 + y^2 - 2xy$$

Common Surfaces

Examples

  • Paraboloid: $$z = x^2 + y^2$$ (bowl)
  • Saddle: $$z = x^2 - y^2$$
  • Hyperboloid: $$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$$

Level Curves (Contours)

Level Sets: f(x,y) = k

For z = x² + y², level curves are circles: x² + y² = k

2D Limits

🚨 LIMIT MUST EXIST FROM ALL DIRECTIONS!

Path-Dependent Limit

$$\lim_{(x,y)→(0,0)} \frac{xy}{x^2+y^2}$$

Along y=0: limit=0
Along y=x: limit=½

Module 5: Partial Derivatives & Gradients

Rates of Change in Multiple Directions

First Partial Derivatives

$$\frac{\partial f}{\partial x} = \lim_{h→0} \frac{f(x+h,y)-f(x,y)}{h}$$

Example: f(x,y) = x²y + xy³

fx = 2xy + y³ (treat y as constant)

fy = x² + 3xy² (treat x as constant)

Clairaut's Theorem

If continuous: $$\frac{∂²f}{∂x∂y} = \frac{∂²f}{∂y∂x}$$

The Gradient Vector

∇f = ⟨fx, fy⟩

Points direction of steepest increase, |∇f| = rate

Directional Derivative

$$D_{\vec{u}}f = \nabla f \cdot \vec{u}$$

where $$\vec{u}$$ is unit vector

Module 6: Multiple Integrals

Volumes and Mass in Higher Dimensions

Double Integral (Volume)

$$\iint_R f(x,y) dA$$

Fubini's Theorem

$$\iint_R f dA = \int_a^b \int_c^d f dy dx = \int_c^d \int_a^b f dx dy$$

Example: Volume Under z = 4 - x - y over [0,1]×[0,1]

$$V = \int_0^1 \int_0^1 (4-x-y) dy dx = \int_0^1 (3.5-x) dx = 3$$

Region Types

Type I: y from g₁(x) to g₂(x)

$$\iint_D f dA = \int_A^B \int_{g₁(x)}^{g₂(x)} f dy dx$$

Polar Coordinates

Jacobian: dx dy = r dr dθ

$$\iint_D f(x,y) dx dy = \iint_{D^\*} f(r\cosθ,r\sinθ) r dr dθ$$

Example: Integral over circle radius a

$$\iint \sqrt{x^2+y^2} dA = \int_0^{2π} \int_0^a r·r dr dθ = \frac{2πa^3}{3}$$

Triple Integrals

$$\iiint_E f dV$$

Computes mass (if f = density) or other 3D quantities

Module 7: Vector Calculus

Line Integrals, Surface Integrals, and Calculus Theorems

🔨 ADVANCED THEORY 🔨

Vector Fields

2D Field: $$\vec{F} = ⟨P(x,y), Q(x,y)⟩$$

Assigns vector to each point

Line Integrals (Work)

$$\int_C \vec{F} · d\vec{r}$$

$$\int_a^b \vec{F}(\vec{r}(t)) · \vec{r}'(t) dt$$

Example: Work along line path

$$\int_C ⟨y, x⟩ · d\vec{r}$$ from (0,0) to (2,0) to (2,3)

Path 1: $$\int_0^2 ⟨0, t⟩ · ⟨1, 0⟩ dt = 0$$

Path 2: $$\int_0^3 ⟨t, 2⟩ · ⟨0, 1⟩ dt = 6$$

Total W = 0 + 6 = 6

Conservative Fields

Fundamental Theorem for Line Integrals

If $$\vec{F} = \nabla f$$: $$\int_C \nabla f · d\vec{r} = f(\vec{r}(b)) - f(\vec{r}(a))$$
Path Independence

Conservative fields' line integrals depend ONLY on endpoints!

Test for Conservative (2D): Py = Qx

Curl and Divergence

Curl: $$\nabla × \vec{F} = ⟨R_y-Q_z, P_z-R_x, Q_x-P_y⟩$$

Measures rotation; zero for conservative

Divergence: $$\nabla · \vec{F} = P_x + Q_y + R_z$$

Measures expansion/compression

Surface Integrals (Flux)

$$\iint_S \vec{F} · d\vec{S} = \iint_D \vec{F} · (\vec{r}_u × \vec{r}_v) du dv$$

Big Theorems

Stokes': $$\oint_C \vec{F} · d\vec{r} = \iint_S (\nabla × \vec{F}) · d\vec{S}$$

Line integral = surface integral of curl

Divergence (Gauss): $$\iint_S \vec{F} · d\vec{S} = \iiint_V (\nabla · \vec{F}) dV$$

Surface flux = volume integral of divergence
