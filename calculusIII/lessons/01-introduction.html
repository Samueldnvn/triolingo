What Makes Calculus III Different?

The key difference between single-variable (Calculus I & II) and multivariable calculus (Calculus III) is the number of dimensions we work in. Here's what changes:

💡 Key Differences: 1D vs 2D/3D
Concept Single-Variable (1D) Multivariable (2D/3D)
Functions y = f(x) z = f(x, y) or multivariable
Graphs Curves in 2D plane Surfaces in 3D space
Derivatives f'(x) (single value) Multiple partial derivatives (∂f/∂x, ∂f/∂y, etc.)
Limits Two directions (left, right) Infinitely many paths to a point
Integrals Area under curve Volume under surface (double/triple integrals)

📐 ESSENTIAL CALCULUS REVIEW

🔨 FOUNDATION CHECK 🔨

Before we dive into multivariable calculus, let's quickly review the key concepts from single-variable calculus that we'll build upon.

1. Derivatives

📐 DERIVATIVE DEFINITION

f'(x) = lim[h→0] [f(x+h) - f(x)]/h

The derivative represents the instantaneous rate of change or the slope of the tangent line.

Common Derivatives to Know:

Function f(x) Derivative f'(x) Notes
xⁿ nxⁿ⁻¹ Power rule
eˣ eˣ Derivative is itself
ln(x) 1/x For x > 0
sin(x) cos(x)
cos(x) -sin(x)
tan(x) sec²(x)

Differentiation Rules:

Product Rule
(fg)' = f'g + fg'
Quotient Rule
(f/g)' = (f'g - fg')/g²
Chain Rule
(f∘g)'(x) = f'(g(x)) · g'(x)

2. Integrals

📐 INTEGRAL DEFINITION

∫f(x) dx = F(x) + C

The integral represents the antiderivative and can represent area under a curve.

Common Integrals to Know:

Function f(x) Integral ∫f(x) dx Notes
xⁿ (n ≠ -1) xⁿ⁺¹/(n+1) + C Power rule for integration
1/x ln|x| + C
eˣ eˣ + C
sin(x) -cos(x) + C
cos(x) sin(x) + C
sec²(x) tan(x) + C
Fundamental Theorem of Calculus
∫ₐᵇ f(x) dx = F(b) - F(a)

where F is any antiderivative of f (F' = f)

Integration Techniques:

💡 u-Substitution (Reverse Chain Rule)

∫f(g(x))g'(x) dx = ∫f(u) du where u = g(x)

Example: ∫2x·cos(x²) dx, let u = x², du = 2x dx
= ∫cos(u) du = sin(u) + C = sin(x²) + C
💡 Integration by Parts
∫u dv = uv - ∫v du

LIATE Rule for choosing u:
L - Logarithmic functions
I - Inverse trigonometric functions
A - Algebraic functions
T - Trigonometric functions
E - Exponential functions

3. Limits

📐 LIMIT DEFINITION

lim[x→a] f(x) = L

As x approaches a, f(x) approaches L. For the limit to exist, the left-hand and right-hand limits must be equal.

Important Limit Properties:

Property Statement
Existence lim[x→a] f(x) exists iff lim[x→a⁺] f(x) = lim[x→a⁻] f(x)
Sum/Difference lim(f ± g) = lim(f) ± lim(g)
Product lim(fg) = lim(f) · lim(g)
Quotient lim(f/g) = lim(f)/lim(g) if lim(g) ≠ 0
Constant Multiple lim(cf) = c · lim(f)

Special Limits:

Two Important Limits
lim[x→0] sin(x)/x = 1
lim[x→∞] (1 + 1/n)ⁿ = e

∞ LIMITS IN TWO DIMENSIONS

🔨 BRIDGE TO MULTIVARIABLE 🔨

This is where Calculus III begins to differ fundamentally from single-variable calculus. Understanding limits in 2D is crucial because everything builds on this foundation.

The Key Difference

⚠️ CRITICAL INSIGHT

In single-variable calculus, when taking a limit as x → a, you only need to check two paths:

  • Approaching from the left: x → a⁻
  • Approaching from the right: x → a⁺

In multivariable calculus, when taking a limit as (x,y) → (a,b), there are infinitely many paths! The limit must be the same along every possible path to the point.

Definition of a Limit in 2D

Limit Definition (2D)
lim[(x,y)→(a,b)] f(x, y) = L

For every ε > 0, there exists δ > 0 such that:
if 0 < √((x-a)² + (y-b)²) < δ, then |f(x, y) - L| < ε

💡 Intuitive Understanding

The limit as (x,y) → (a,b) is L if f(x, y) gets arbitrarily close to L as (x, y) approaches (a, b) from any direction.

Proving a Limit Does NOT Exist

To show that a limit does not exist, you only need to find two different paths that give different limit values. If even two paths disagree, the overall limit cannot exist.

🔧 Common Paths to Test

When testing limits as (x,y) → (0,0), these are the first paths to try:

  • Along x-axis: Set y = 0, take limit as x → 0
  • Along y-axis: Set x = 0, take limit as y → 0
  • Along line y = x: Set y = x, take limit as x → 0
  • Along line y = kx: Set y = kx, take limit as x → 0 (any constant k)
  • Along parabola: Set y = x², take limit as x → 0
  • Along curves: Any curve passing through (0,0)
💡 Example: Limit Does Not Exist

Does lim[(x,y)→(0,0)] (x² - y²)/(x² + y²) exist?

Path 1: y = 0 (x-axis)
(x² - 0)/(x² + 0) = x²/x² = 1 → limit = 1

Path 2: x = 0 (y-axis)
(0 - y²)/(0 + y²) = -y²/y² = -1 → limit = -1

Conclusion: Since 1 ≠ -1, the limit DOES NOT EXIST!
💡 Example: Limit Exists

Does lim[(x,y)→(0,0)] (x + y) exist?

Path 1: y = 0
x + 0 = x → limit = 0

Path 2: x = 0
0 + y = y → limit = 0

Path 3: y = x
x + x = 2x → limit = 0

Path 4: y = kx
x + kx = (1+k)x → limit = 0

All paths give 0, so the limit exists and equals 0!
⚠️ IMPORTANT WARNING

Even if several paths give the same limit, you CANNOT conclude the limit exists! There are infinitely many other paths. To PROVE a limit exists, you must rigorously show it's the same for ALL paths (typically using the ε-δ definition or squeeze theorem).

Continuity in 2D

Continuity Definition

f is continuous at (a, b) if:

lim[(x,y)→(a,b)] f(x, y) = f(a, b)

This means three conditions must be met:
1. f(a, b) is defined
2. The limit exists
3. The limit equals the function value
