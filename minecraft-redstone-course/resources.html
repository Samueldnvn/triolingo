Universal Gates

Both NAND and NOR are universal gates - you can build ANY other logic gate using only NAND or only NOR gates. This is important for understanding computer architecture and circuit optimization.

Calculating Two's Complement

To convert +5 to -5 in 4-bit: Step 1: Write +5 in binary 0101 Step 2: Invert all bits 1010 Step 3: Add 1 1010 + 1 ------- 1011 ← This is -5 To check: 8 + 0 + 2 + 1 = 11? No, MSB is negative -(8-0+2+1) = -(8-3) = -5 ✓

Flag Conditions

  • Zero Flag (Z): Result == 0? OR all output bits, invert
  • Carry Flag (C): Last adder's carry-out
  • Overflow Flag (V): XOR of last two carry bits
  • Sign Flag (S): Most significant bit (MSB)

Quick Cheatsheet

Essential commands and formulas for quick reference.

Boolean Identities

Identity: X OR 0 = X X AND 0 = 0 X OR 1 = 1 X AND 1 = X X OR X = X X AND X = X NOT(NOT X) = X Commutative: X OR Y = Y OR X X AND Y = Y AND X Associative: (X OR Y) OR Z = X OR (Y OR Z) (X AND Y) AND Z = X AND (Y AND Z) Distributive: X AND (Y OR Z) = (X AND Y) OR (X AND Z) De Morgan's: NOT(X AND Y) = NOT X OR NOT Y NOT(X OR Y) = NOT X AND NOT Y

Binary Arithmetic Rules

Addition: 0+0=0 (carry 0) 0+1=1 (carry 0) 1+0=1 (carry 0) 1+1=0 (carry 1) 1+1+1=1 (carry 1) Subtraction: 0-0=0 1-0=1 1-1=0 0-1=1 (borrow 1) Multiplication (shift-add): Multiply by powers of 2 = shift left Divide by powers of 2 = shift right

Minecraft Redstone Tips

  • Signal Strength: 0 = OFF, 1-15 = ON. Treat any signal as binary 1.
  • Repeater Delay: 1-4 ticks. Controls timing and prevents signal decay.
  • Comparator: Has comparison mode (subtract front from side) and subtraction mode.
  • Compact Layout: Build in 3D! Use vertical space for signal routing.
  • Cable Management: Color-code using different block materials for clarity.
  • Debugging: Place lamps at key points to visualize signal flow.
  • Save Often: Redstone computers are complex - save after each milestone.
