❓ FREQUENTLY ASKED QUESTIONS

How long will this course take?
The course is designed for approximately 40-60 hours of hands-on building. Each module takes 4-8 hours for practical application. However, you can go at your own pace!
Do I need any mods?
No! This course uses vanilla Minecraft redstone only. Creative mode is recommended for ease of building, but everything works in survival too.
What version of Minecraft?
The concepts work on any version from Java 1.5+ or Bedrock 0.13+. Modern versions (1.18+) are recommended for best performance.
Will my computer be Turing Complete?
Yes! With sufficient memory, the computer you build will be Turing Complete and capable of running any computable program.
