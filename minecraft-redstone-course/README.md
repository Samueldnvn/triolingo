# Redstone Computing Academy - Complete Course

A comprehensive, heavily Minecraft-themed course on building redstone computers from scratch using vanilla Minecraft redstone.

## 📚 Course Overview

This course teaches everything needed to build an 8-bit computer in Minecraft, covering:

- **Logic Gates**: NOT, AND, OR, NAND, NOR, XOR, XNOR
- **Advanced Circuitry**: Adders, MUX, Decoders, Clocks
- **Binary Systems**: Conversion, arithmetic, two's complement, hexadecimal
- **ALU Design**: Arithmetic and logic operations, flags
- **Memory Architecture**: Flip-flops, registers, RAM, ROM
- **Control Unit**: Instruction sets, fetch-decode-execute cycle
- **I/O Systems**: Input panels, hex displays, storage
- **Full Computer**: Complete integration and programming

## 🎯 Course Structure

### Module 1: Logic Gates
- 6 basic gates with truth tables
- Minecraft implementations for each
- Boolean algebra and optimization
- 2-bit comparator project

### Module 2: Advanced Circuitry
- Half and full adders
- Ripple-carry adder
- Multiplexers and demultiplexers
- Decoders and encoders
- Clock generators
- 4-bit calculator project

### Module 3: Binary Systems
- Binary to decimal conversion
- Binary arithmetic
- Two's complement (signed numbers)
- Hexadecimal conversion
- Binary calculator project

### Module 4: ALU Design
- 16 operations (add, subtract, bitwise ops, etc.)
- MUX-based output selection
- Flag generation (Zero, Carry, Overflow, Sign)
- 4-bit ALU project

### Module 5: Memory Architecture
- RS, JK, D, T flip-flops
- Registers
- RAM and ROM design
- Address decoding
- 4-bit RAM project

### Module 6: Control Unit
- Instruction set (16 instructions)
- Instruction decoder
- Fetch-decode-execute cycle
- Program counter
- Conditional jumps
- 8-instruction control unit project

### Module 7: I/O Systems
- Button input panels
- Hex display systems
- Data persistence (item-based)
- I/O address mapping

### Module 8: Full Computer
- Complete 8-bit computer integration
- Bus design
- Clock distribution
- Test programs
- Complete computer project (FINAL PROJECT)

### Module 9: Advanced Topics
- Pipelining
- Interrupt handling
- CPU cache concepts
- Graphics systems

### Module 10: Final Project
- Complete 8-bit computer with all components
- 256 bytes ROM + 256 bytes RAM
- Multiple test programs
- Extra credit opportunities
