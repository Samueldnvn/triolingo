📐 VECTORS & 3D SPACE

🔨 FOUNDATION WORK: VECTORS 🔨

Let's start by understanding vectors - quantities that have both magnitude and direction.

What is a Vector?

📐 VECTOR DEFINITION

A vector v in 3D space can be written as:

v = 〈v₁, v₂, v₃〉 = v₁i + v₂j + v₃k

Where:

  • i, j, k are unit vectors in the x, y, z directions
  • v₁, v₂, v₃ are the components
3D Coordinate System

Vector Operations

Vector Addition & Scalar Multiplication

If u = 〈u₁, u₂, u₃〉 and v = 〈v₁, v₂, v₃〉, then:

u + v = 〈u₁+v₁, u₂+v₂, u₃+v₃〉

cu = 〈cu₁, cu₂, cu₃〉 (where c is a scalar)

🔧 Practice: Vector Operations

Dot Product

The Dot Product

u · v = |u||v|cos(θ) = u₁v₁ + u₂v₂ + u₃v₃

Where θ is the angle between vectors

💡 Key Property: Dot Product and Orthogonality

Two vectors are orthogonal (perpendicular) if and only if their dot product is zero.

u ⟂ v ⇔ u · v = 0
🔧 Practice: Dot Product

Are u = 〈1, 0, 0〉 and v = 〈0, 5, 0〉 orthogonal?

Cross Product

The Cross Product

u × v = |u||v|sin(θ)n

The cross product gives a vector perpendicular to both u and v.

⚠️ IMPORTANT: Cross Product Properties
  • u × v = -(v × u) (NOT commutative!)
  • |u × v| = |u||v|sin(θ) gives the area of the parallelogram
  • i × j = k, j × k = i, k × i = j

🔄 VECTOR FUNCTIONS

🔨 BUILDING IN MOTION: VECTOR FUNCTIONS 🔨

A vector function maps real numbers to vectors. These are essential for describing motion in 3D space.

Definition & Notation

Vector Function

r(t) = 〈f(t), g(t), h(t)〉 = f(t)i + g(t)j + h(t)k

📐 INTERPRETATION

The vector function r(t) represents:

  • r(t) = position of a particle at time t
  • The set of all points r(t) = space curve
  • r'(t) = velocity vector
  • r''(t) = acceleration vector

Differentiation of Vector Functions

Derivative of a Vector Function

r'(t) = 〈f'(t), g'(t), h'(t)〉

We differentiate each component separately!

💡 Example: Finding Velocity and Acceleration

Given r(t) = 〈t², sin(t), eᵗ〉:

v(t) = r'(t) = 〈2t, cos(t), eᵗ〉
a(t) = r''(t) = 〈2, -sin(t), eᵗ〉
🔧 Practice: Vector Function Derivatives
Find r'(t) for r(t) = 〈t³, 2t, cos(t)〉

Arc Length

Arc Length of a Space Curve

L = ∫[a,b] |r'(t)| dt = ∫[a,b] √(f'(t)² + g'(t)² + h'(t)²) dt

💡 Example: Finding Arc Length

Find the length of r(t) = 〈cos(t), sin(t), t〉 for 0 ≤ t ≤ 2π

r'(t) = 〈-sin(t), cos(t), 1〉
|r'(t)| = √(sin²(t) + cos²(t) + 1) = √2
L = ∫₀²π √2 dt = 2π√2
Helix: r(t) = 〈cos(t), sin(t), t〉

Curvature

Curvature

κ(t) = |r'(t) × r''(t)| / |r'(t)|³

Curvature measures how sharply a curve bends.

⚠️ Key Insight
  • Straight lines have κ = 0
  • Circles have constant curvature κ = 1/radius
  • Large curvature = sharp turn

📊 MULTIVARIABLE FUNCTIONS

🔨 BUILDING UP: FUNCTIONS OF SEVERAL VARIABLES 🔨

Now we're moving from curves to surfaces. Functions of multiple variables map from ℝⁿ to ℝ.

Functions of Two Variables

Function of Two Variables

z = f(x, y)

This represents a surface in 3D space!

📐 DOMAIN & RANGE

  • Domain: All (x, y) where f(x, y) is defined
  • Range: All possible z = f(x, y) values
  • Domain is a region in the xy-plane
  • Graph is a surface in 3D space
Surface: z = x² + y² (Paraboloid)

Level Curves (Contour Maps)

Level Curves

Level curves: f(x, y) = k (for constant k)

These are 2D "slices" of the surface at height k.

💡 Example: Level Curves of Paraboloid

For f(x, y) = x² + y²:

  • k = 0: Point at origin
  • k = 1: Circle x² + y² = 1 (radius 1)
  • k = 4: Circle x² + y² = 4 (radius 2)
  • k = 9: Circle x² + y² = 9 (radius 3)
Level Curves (Contour Map)

Limits and Continuity

Limit in Two Variables

lim[(x,y)→(a,b)] f(x, y) = L

⚠️ CRITICAL DIFFERENCE from Single Variable

In 2D, you can approach a point from infinitely many directions. The limit must be the same from all paths!

💡 Testing for Existence of Limits

To show a limit doesn't exist: Find two different paths with different limit values.

Common paths to try:

  • Along x-axis (y = 0)
  • Along y-axis (x = 0)
  • Along line y = x
  • Along x = ky (any line through origin)

∂ PARTIAL DERIVATIVES

🔨 FINE DETAIL WORK: PARTIAL DERIVATIVES 🔨

Partial derivatives extend differentiation to functions of multiple variables. We differentiate with respect to one variable while treating others as constants.

Definition

Partial Derivatives

∂f/∂x = lim[h→0] [f(x+h, y) - f(x, y)]/h

∂f/∂y = lim[h→0] [f(x, y+h) - f(x, y)]/h

📐 HOW TO COMPUTE

To find ∂f/∂x:

  • Treat y as a constant
  • Differentiate with respect to x normally
  • Use all the usual rules!

To find ∂f/∂y:

  • Treat x as a constant
  • Differentiate with respect to y normally
  • Use all the usual rules!
💡 Example: Computing Partial Derivatives

Find ∂f/∂x and ∂f/∂y for f(x, y) = x²y + 3xy² + 5

∂f/∂x: Treat y as constant → 2xy + 3y²
∂f/∂y: Treat x as constant → x² + 6xy
Tangent Planes

Higher Order Partial Derivatives

Second Order Partial Derivatives

∂²f/∂x² = fₓₓ, ∂²f/∂y² = f_yy

∂²f/∂x∂y = fₓy, ∂²f/∂y∂x = f_yₓ

⚠️ CLAIRAUT'S THEOREM

Clairaut's Theorem: If the mixed partial derivatives are continuous, then fₓy = f_yₓ.

Order usually doesn't matter for mixed partials!

The Gradient

Gradient Vector

∇f = 〈∂f/∂x, ∂f/∂y, ∂f/∂z〉

💡 Key Properties of the Gradient
  • ∇f points in the direction of steepest ascent
  • -∇f points in the direction of steepest descent
  • |∇f| gives the rate of steepest ascent
  • ∇f is perpendicular to level curves

Directional Derivatives

Directional Derivative

Dᵤf = ∇f · u

Where u is a unit vector pointing in the direction of interest.

∫ MULTIPLE INTEGRALS

🔨 BUILDING VOLUME: MULTIPLE INTEGRALS 🔨

Multiple integrals let us integrate over regions in 2D, 3D, and beyond. They're used to find areas, volumes, masses, and more.

Double Integrals

Double Integral

∬ᴿ f(x, y) dA

📐 INTERPRETATION

  • If f(x, y) = 1: Gives the area of region R
  • If f(x, y) > 0: Gives the volume under the surface
  • If f(x, y) = density: Gives the mass of the region
Fubini's Theorem (Iterated Integrals)

∬ᴿ f(x, y) dA = ∫[a,b] ∫[g₁(x),g₂(x)] f(x, y) dy dx

= ∫[c,d] ∫[h₁(y),h₂(y)] f(x, y) dx dy

Order of integration can be swapped!

💡 Example: Computing a Double Integral

Evaluate ∬ᴿ (x + 2y) dA where R = [0, 2] × [0, 1]

∫₀² ∫₀¹ (x + 2y) dy dx = ∫₀² [xy + y²]ᵧ₌₀¹ dx
= ∫₀² (x + 1) dx = [x²/2 + x]₀² = 2 + 2 = 4

Triple Integrals

Triple Integral

∭ᴱ f(x, y, z) dV

💡 Applications of Triple Integrals
  • Volume: ∭ᴱ 1 dV
  • Mass: ∭ᴱ ρ(x, y, z) dV (where ρ is density)
  • Average value: (1/volume) ∭ᴱ f dV
  • Center of mass: x̄ = (1/mass) ∭ᴱ xρ dV

Change of Variables

Jacobian Determinant

dA = |J| du dv, where |J| = |∂(x, y)/∂(u, v)|

|J| = |[∂x/∂u ∂x/∂v; ∂y/∂u ∂y/∂v]|

⚠️ POPULAR COORDINATE SYSTEMS
  • Polar coordinates: x = r cos(θ), y = r sin(θ), |J| = r
  • Cylindrical: x = r cos(θ), y = r sin(θ), z = z, |J| = r
  • Spherical: x = ρ sin(φ) cos(θ), y = ρ sin(φ) sin(θ), z = ρ cos(φ), |J| = ρ² sin(φ)
Polar Coordinates

⃗ VECTOR CALCULUS

🔨 FINAL CONSTRUCTION: VECTOR CALCULUS 🔨

Vector calculus combines vector operations with integration. It's the foundation for electromagnetism, fluid dynamics, and more.

Line Integrals

Line Integral of Scalar Field

∫ᶜ f(x, y, z) ds = ∫[a,b] f(r(t)) |r'(t)| dt

Line Integral of Vector Field: ∫ᶜ F · dr = ∫[a,b] F(r(t)) · r'(t) dt

📐 INTERPRETATION

  • Scalar: "Add up" values along the curve
  • Vector: Work done by force field along path
  • Depends on direction (orientation) of curve

Fundamental Theorem for Line Integrals

FTLI (Conservative Fields)

∫ᶜ ∇f · dr = f(r(b)) - f(r(a))

If F = ∇f (conservative), the integral depends only on endpoints!

Green's Theorem

Green's Theorem (2D)

∮ᶜ P dx + Q dy = ∬ᴿ (∂Q/∂x - ∂P/∂y) dA

Converts line integral around closed curve to double integral over region

Stokes' Theorem

Stokes' Theorem (3D Surfaces)

∮ᶜ F · dr = ∬ₛ curl F · dS

Relates line integral around boundary to surface integral of curl

Divergence Theorem

Divergence Theorem (Gauss)

∬ₛ F · dS = ∭ᴱ div F dV

Converts surface integral to volume integral

⚠️ KEY RELATIONSHIPS
  • div (divergence): Measures "outflow" at a point, scalar
  • curl: Measures "rotation" at a point, vector
  • curl ∇f = 0 (gradient fields are irrotational)
  • div curl F = 0 (curl fields are incompressible)
