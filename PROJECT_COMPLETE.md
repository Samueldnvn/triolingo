## Why HTML Not MP4?

**Technical limitation:**
- MP4 video creation requires `ffmpeg`
- `ffmpeg` is not installed on this system
- Cannot install without admin/sudo privileges
- System-level package installation blocked

**Solution provided:**
- HTML presentation works immediately
- Same viewing experience as video
- Smoother transitions and interactivity
- Works on any device with a web browser
- No installation required

**To create MP4 later:**
```bash
# Option 1: Install ffmpeg (needs admin)
sudo apt-get install ffmpeg
python3 assemble_bill_gates_mp4.py

# Option 2: Use online video converter
# Upload audio/ and images/ to generate MP4
```
