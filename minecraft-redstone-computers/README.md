# Minecraft Redstone Computers Course

A complete, comprehensive course on building computers from scratch using Minecraft redstone.

## Course Overview

This course takes you from understanding basic logic gates to building fully functional computer processors in Minecraft. Through in-depth lessons, hundreds of practice problems, and hands-on projects, you'll master the art of digital logic design and computer architecture—all within the blocky world of Minecraft.

## Prerequisites

- Solid understanding of Minecraft redstone mechanics (how redstone signals propagate, redstone dust, redstone blocks, comparators, repeaters)
- No prior knowledge of computer science or digital logic required
- Familiarity with basic Minecraft building

## What You'll Learn

- **Module 1: Logic Gates in Minecraft** - Building AND, OR, NOT, XOR, NAND, and NOR gates; understanding truth tables and logical equivalence
- **Module 2: Adders and Arithmetic Circuits** - Half adders, full adders, ripple carry adders, binary arithmetic in Minecraft
- **Module 3: Memory and Storage** - RS latches, D flip-flops, T flip-flops, JK flip-flops, registers, RAM cells
- **Module 4: Clocks and Timing** - Redstone clocks, pulse generators, pulse extenders, synchronization circuits
- **Module 5: Data Transmission Systems** - Busses, data cables, repeaterless transmission, wireless redstone, multiplexing
- **Module 6: The Arithmetic Logic Unit (ALU)** - Designing and building ALUs, adder-subtractor circuits, comparison logic, shift operations
- **Module 7: Control Units and Decoders** - Instruction decoders, control logic, state machines, microcode basics
- **Module 8: Complete CPU Architecture** - Instruction sets, fetch-decode-execute cycle, program counters, complete processor design
- **Final: Advanced Computer Design** - Optimizing designs, pipelining concepts, complex ALUs, advanced memory systems

## Getting Started

Open `index.html` in your web browser to begin the course. Progress will be automatically saved to your local storage.

## Course Structure

Each module includes:
1. **Introduction** with learning objectives
2. **Detailed explanations** of concepts and mechanics
3. **Multiple examples** with step-by-step Minecraft builds
4. **Practice problems** (50-80 per module) progressing from easy to hard
5. **Module quiz** (8-15 questions, 70% required to pass)

## Mastery Standards

To complete this course with mastery:
- Pass all 8 module quizzes (70%+ each)
- Score 90%+ on the final exam
- Complete the final project: Build a working CPU capable of executing simple programs

## Minecraft Version

This course assumes Minecraft Java Edition 1.20+ but concepts apply to all recent versions. Some advanced features may use the most current redstone mechanics.
