# 🚀 Quick Start Guide

## Option 1: Run Immediately (Web Browser)

1. **Open the app:**
   - Just double-click `index.html` to open in your browser
   - OR right-click → Open with → Your favorite browser

2. **Start practicing:**
   - Set sequence length (starts at 8)
   - Choose type (Numbers, Letters, or Mixed)
   - Click "Start Practice"
   - Click "Show Sequence" to see what to memorize
   - Click "Start Recall" when ready
   - Type what you remember!

That's it! No installation needed.

---

## Option 2: Run as Desktop App

### Step 1: Install Node.js
1. Go to https://nodejs.org
2. Download and install the LTS version (recommended for most users)

### Step 2: Install Dependencies
Open terminal/command prompt in this folder and run:
```bash
npm install
```

This will take 30-60 seconds.

### Step 3: Run the App
```bash
npm start
```

The app will open in a window with all features available!

---

## Option 3: Create Desktop Shortcut (Linux/WSL)

Run this command:
```bash
chmod +x create-shortcut.sh
./create-shortcut.sh
```

This creates a shortcut on your desktop. Double-click it to launch!

---

## 🎯 First Practice Session

1. **Default settings are perfect for beginners:**
   - Length: 8 characters
   - Numbers only
   - Untimed (so you can take your time)

2. **Click "Start Practice"**

3. **Memorize:**
   - Click "Show Sequence"
   - You'll see: `8 5 2 9 1 4 7 3`
   - Use whatever memory technique works for you
   - When you're ready, click "Start Recall"

4. **Type what you remember:**
   - Type: `85291473`
   - Press Enter or click "Submit"

5. **See your results!**
   - Correct or incorrect
   - Time taken
   - Click "New Round" to try again

## 📝 Practice with Custom Sequences

Want to practice something specific?

1. Go to "Custom Sequences"
2. Enter a name: "Japanese Kana"
3. Enter characters: `ka, ki, ku, ke, ko, ma, mi, mu, me, mo`
4. Click "Add Sequence"
5. Click "Use" next to it
6. Practice with that sequence!

Great for:
- Syllables (Japanese, Korean, etc.)
- Special characters
- Subset of numbers
- Any custom character set!

## 💡 Tips for Beginners

1. **Start easy:**
   - Begin with 4-5 character sequences
   - Use numbers only
   - Untimed mode

2. **Gradually increase:**
   - Add 1 more character when you're comfortable
   - Try letters or mixed
   - Add time limits once confident

3. **Use memory palace techniques:**
   - Visualize your home
   - Place items in specific locations
   - Walk through your palace mentally

4. **Practice regularly:**
   - 10-15 minutes daily is great
   - Consistency beats cramming

## 🔧 Troubleshooting

**Q: Sounds don't play**
- A: Click anywhere in the app first (browsers need user interaction)

**Q: Can't type in the answer box**
- A: Make sure you clicked "Start Recall"

**Q: Want bigger text**
- A: Go to Settings → Font Size → Large

**Q: Dark mode?**
- A: Go to Settings → Theme → Dark
