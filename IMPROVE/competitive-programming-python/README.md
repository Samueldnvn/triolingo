# Competitive Programming with Python - Complete Course

## Overview
A comprehensive competitive programming course covering all data structures, algorithms, and patterns needed for LeetCode and coding interviews.

## Target Audience
- People who already know Python basics
- From beginner to expert level
- Preparing for coding interviews and competitive programming

## Course Structure

### 10 Modules
1. **Foundations & Prerequisites** - Complexity analysis, Python optimization, algorithm design
2. **Data Structures** - Arrays, strings, linked lists, stacks, queues, hash tables, trees, graphs
3. **Hash Tables Pattern** - When to use hash tables, common patterns, frequency counting, subarray sum, anagrams
4. **Sliding Window Pattern** - Fixed and variable-size windows, longest substring, minimum window substring
5. **Two Pointers Pattern** - Left-right pointers, slow-fast pointers, cycle detection, middle finding
6. **Binary Search Pattern** - Standard binary search, binary search on answer, rotated arrays, boundary finding
7. **Dynamic Programming** - 1D DP, 2D DP, knapsack problems, LCS, edit distance, advanced DP (bitmask)
8. **Greedy Algorithms** - Greedy fundamentals, activity selection, Dijkstra, Prim, Kruskal
9. **Graph Algorithms** - BFS, DFS, shortest paths (Dijkstra, Bellman-Ford, Floyd-Warshall), MST, Union-Find
10. **Advanced Topics** - Segment trees, Fenwick trees, Tries, KMP, backtracking, bitmask DP, problem-solving strategies
