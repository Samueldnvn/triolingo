Why Hash Tables Pattern is Essential

Hash tables are the unsung heroes of competitive programming. Many problems that seem to require nested loops (O(n^2)) can be solved in O(n) or O(n log n) using hash tables. The key is recognizing when fast lookups or storing additional information can optimize your solution.

THE MAGIC OF HASH TABLES

Hash tables provide O(1) average time for insert, delete, and lookup operations. By storing previously seen values or computing frequencies, you can transform quadratic algorithms into linear ones. This pattern appears frequently in LeetCode problems and competitive programming contests.

When to Use Hash Tables

Scenario Use Hash Table Reason
Need to check if element exists Yes (set or dict) O(1) vs O(n) for list/array
Need to count frequencies Yes (dict/Counter) Efficient counting in one pass
Need to lookup by key Yes (dict) O(1) vs O(log n) for binary search
Elements are sorted and you search No (binary search better) O(log n) with no extra space
Small dataset (n < 100) Maybe Overhead might not be worth it
Need to maintain order of insertion No (use list or OrderedDict) Dict doesn't preserve order in older Python

Set vs Dictionary: Choosing the Right Tool

DECISION GUIDE
Requirement Use This Example
Store key-value pairs dict word -> count, number -> index
Membership testing only set seen elements, duplicates check
Need to access value by key dict Two Sum complement lookup
Need unique elements set remove duplicates, intersection

Pattern Recognition

Identifying when hash tables are the optimal solution

Problem Indicators for Hash Tables

Recognizing when to use hash tables is a crucial skill. Watch for these indicators in problem statements:

SIGNALS FOR HASH TABLES
  • "Find if two numbers sum to target": Two Sum pattern - use dict for complement lookup
  • "Group elements by some property": Grouping pattern - use dict to map property to group
  • "Count occurrences/frequency": Frequency pattern - use dict or Counter
  • "First element to appear k times": Frequency with ordering - dict
  • "Find duplicate elements": Set or frequency dict
  • "Check if any value appears at least twice": Set for seen tracking
  • "Find elements in common between two arrays": Set intersection
  • "Subarray with sum k": Prefix sum with dict
  • "Longest substring without repeating characters": Set/dict for tracking
  • "Find first non-repeating character": Two-pass frequency counting

Common Mistakes and When NOT to Use Hash Tables

PITFALLS TO AVOID
  • Using hash table on sorted data: If data is sorted, binary search might be better
  • Excessive memory usage: Hash tables use O(n) extra space, which might not be acceptable
  • Collisions: In rare cases, hash function can cause O(n) instead of O(1)
  • Wrong key choice: Choosing a non-hashable key or wrong key for the problem
  • Using list for membership testing: Use set instead for O(1) lookups
  • Not using built-in Counter: Counter is optimized for counting

Pattern Recognition Examples

Example 1: Recognizing Two Sum Pattern

# Problem: Given an array of integers, return indices of two numbers that add up to target
# Pattern Recognition Indicators:
# - "Two numbers that add up to target"
# - Need to find complement (target - current)
# - Need to return indices (key-value: number -> index)

# O(n) solution using hash table
def two_sum(nums, target):
    """
    Two Sum pattern
    Time: O(n), Space: O(n)
    """
    seen = {}  # number -> index
    
    for i, num in enumerate(nums):
        complement = target - num
        
        if complement in seen:
            return [seen[complement], i]
        
        seen[num] = i
    
    return None  # No solution

# Example
nums = [2, 7, 11, 15]
target = 9
print(two_sum(nums, target))  # [0, 1]

Example 2: Recognizing Grouping Pattern

# Problem: Group anagrams together
# Pattern Recognition Indicators:
# - "Group strings by some property" (anagrams)
# - "Organize elements"
# - Property can be derived from each element (sorted characters as key)

from collections import defaultdict

def group_anagrams(words):
    """
    Group anagrams pattern
    Time: O(n * k log k) where n = words, k = max word length
    Space: O(n * k)
    """
    groups = defaultdict(list)  # sorted_word -> list of words
    
    for word in words:
        # Key is sorted characters (anagrams have same sorted form)
        key = ''.join(sorted(word))
        groups[key].append(word)
    
    return list(groups.values())

# Example
words = ["eat", "tea", "tan", "ate", "nat", "bat"]
print(group_anagrams(words))
# [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]

Example 3: Recognizing Frequency Pattern

# Problem: Find the element that appears more than n/2 times
# Pattern Recognition Indicators:
# - "Find element that appears X times"
# - "Majority element"
# - "Frequency counting"

from collections import Counter

def majority_element(nums):
    """
    Frequency counting pattern
    Time: O(n), Space: O(n)
    """
    freq = Counter(nums)
    n = len(nums)
    
    for num, count in freq.items():
        if count > n // 2:
            return num
    
    return None

# O(1) space solution (Boyer-Moore Voting Algorithm)
def majority_element_o1(nums):
    candidate = None
    count = 0
    
    for num in nums:
        if count == 0:
            candidate = num
            count = 1
        elif num == candidate:
            count += 1
        else:
            count -= 1
    
    return candidate

# Example
nums = [3, 2, 3]
print(majority_element(nums))  # 3

Counting Frequencies

The most common hash table pattern

Frequency Counting Fundamentals

Counting frequencies is one of the most common hash table applications. The pattern is simple: iterate through the data once, counting occurrences of each element using a dictionary or Counter.

Method 1: Manual Dictionary Counting

def count_frequencies(arr):
    """
    Count frequencies manually using dictionary
    Time: O(n), Space: O(k) where k = unique elements
    """
    freq = {}
    
    for item in arr:
        freq[item] = freq.get(item, 0) + 1
    
    return freq

# Alternative using setdefault
def count_frequencies_setdefault(arr):
    freq = {}
    for item in arr:
        freq.setdefault(item, 0)
        freq[item] += 1
    return freq

# Alternative using defaultdict
from collections import defaultdict

def count_frequencies_defaultdict(arr):
    freq = defaultdict(int)
    for item in arr:
        freq[item] += 1
    return dict(freq)

# Example
arr = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
print(count_frequencies(arr))
# {1: 1, 2: 2, 3: 3, 4: 4}

Method 2: Using collections.Counter

from collections import Counter

def count_frequencies_counter(arr):
    """
    Count frequencies using Counter (most Pythonic)
    Time: O(n), Space: O(k)
    """
    return Counter(arr)

# Counter has additional useful methods
def frequency_operations(arr):
    freq = Counter(arr)
    
    print("Most common:", freq.most_common(3))  # Top 3
    print("Total count:", sum(freq.values()))   # Total elements
    print("Unique count:", len(freq))          # Unique elements
    print("Elements with count > 1:", [k for k, v in freq.items() if v > 1])
    
    # Update with another iterable
    freq.update([1, 2, 2])
    
    # Subtract counts
    freq.subtract([1, 2])
    
    return freq

# Example
arr = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
freq = count_frequencies_counter(arr)
print(freq)  # Counter({4: 4, 3: 3, 2: 2, 1: 1})

1. Find Duplicates

def find_duplicates(arr):
    """
    Find all elements that appear more than once
    Time: O(n), Space: O(n)
    """
    freq = Counter(arr)
    return [item for item, count in freq.items() if count > 1]

def has_duplicate(arr):
    """
    Check if any duplicate exists
    Time: O(n), Space: O(n)
    """
    seen = set()
    for item in arr:
        if item in seen:
            return True
        seen.add(item)
    return False

# More efficient: check while building freq
def find_first_duplicate(arr):
    """
    Find first duplicate (first element to appear twice)
    Time: O(n), Space: O(n)
    """
    seen = set()
    for item in arr:
        if item in seen:
            return item
        seen.add(item)
    return None

# Example
arr = [1, 2, 3, 2, 4, 3, 5]
print(find_duplicates(arr))      # [2, 3]
print(find_first_duplicate(arr)) # 2

2. Find K Most Frequent Elements

from collections import Counter
import heapq

def top_k_frequent(arr, k):
    """
    Find k most frequent elements
    Time: O(n log k), Space: O(k)
    """
    freq = Counter(arr)
    # Use min-heap of size k
    return heapq.nlargest(k, freq.keys(), key=freq.get)

def top_k_frequent_bucket(arr, k):
    """
    Find k most frequent using bucket sort
    Time: O(n), Space: O(n)
    """
    freq = Counter(arr)
    max_freq = max(freq.values())
    
    # Bucket sort by frequency
    buckets = [[] for _ in range(max_freq + 1)]
    for num, count in freq.items():
        buckets[count].append(num)
    
    # Collect from highest frequency
    result = []
    for i in range(max_freq, 0, -1):
        for num in buckets[i]:
            result.append(num)
            if len(result) == k:
                return result
    
    return result

# Example
arr = [1, 1, 1, 2, 2, 3]
k = 2
print(top_k_frequent(arr, k))      # [1, 2]
print(top_k_frequent_bucket(arr, k))  # [1, 2]

3. First Unique Character in String

def first_unique_character(s):
    """
    Find first non-repeating character
    Time: O(n), Space: O(1) - fixed alphabet size
    """
    freq = Counter(s)
    
    for i, char in enumerate(s):
        if freq[char] == 1:
            return i  # Return index
    
    return -1  # No unique character

def first_unique_character_value(s):
    """Return the character itself"""
    freq = Counter(s)
    
    for char in s:
        if freq[char] == 1:
            return char
    
    return None

# Example
s = "leetcode"
print(first_unique_character(s))       # 0 (character 'l')
print(first_unique_character_value(s)) # 'l'

4. Check if All Characters Have Equal Frequency

def are_occurrences_equal(s):
    """
    Check if all characters appear the same number of times
    Time: O(n), Space: O(k) where k = unique characters
    """
    freq = Counter(s)
    frequencies = set(freq.values())
    return len(frequencies) == 1

def min_deletions_for_equal_freq(s):
    """
    Find minimum deletions to make all frequencies equal
    Time: O(n), Space: O(k)
    """
    freq = Counter(s)
    freq_values = list(freq.values())
    freq_values.sort(reverse=True)
    
    deletions = 0
    target = freq_values[0]
    
    for i in range(1, len(freq_values)):
        if freq_values[i] == target:
            continue
        elif freq_values[i] > target:
            deletions += freq_values[i] - target
        elif freq_values[i] < target:
            if target == 1:
                deletions += freq_values[i]
            else:
                deletions += freq_values[i]
                target -= 1
    
    return deletions

# Example
s = "abacbc"
print(are_occurrences_equal(s))  # True (a=2, b=2, c=2)

String-Specific Frequency Problems

5. Valid Anagram

def is_anagram(s1, s2):
    """
    Check if two strings are anagrams
    Time: O(n), Space: O(k) where k = alphabet size
    """
    if len(s1) != len(s2):
        return False
    
    return Counter(s1) == Counter(s2)

def is_anagram_manual(s1, s2):
    """
    Manual version for space optimization
    Time: O(n), Space: O(k)
    """
    if len(s1) != len(s2):
        return False
    
    freq = [0] * 26  # For lowercase English letters
    
    for c1, c2 in zip(s1, s2):
        freq[ord(c1) - ord('a')] += 1
        freq[ord(c2) - ord('a')] -= 1
    
    return all(f == 0 for f in freq)

# Example
print(is_anagram("anagram", "nagaram"))  # True
print(is_anagram("rat", "car"))          # False

6. Ransom Note

def can_construct(ransom_note, magazine):
    """
    Check if ransom note can be constructed from magazine
    Time: O(n + m), Space: O(k)
    """
    magazine_freq = Counter(magazine)
    
    for char in ransom_note:
        if magazine_freq[char] <= 0:
            return False
        magazine_freq[char] -= 1
    
    return True

# Example
ransom_note = "a"
magazine = "b"
print(can_construct(ransom_note, magazine))  # False

ransom_note = "aa"
magazine = "aab"
print(can_construct(ransom_note, magazine))  # True
PRO TIP: For problems involving only lowercase English letters, use a frequency array of size 26 instead of a dictionary. This is faster and uses less memory as the size is fixed.

Complements Pattern

Finding pairs and combinations efficiently

Understanding the Complements Pattern

The complements pattern is used when you need to find elements that satisfy a specific relationship based on arithmetic operations (addition, subtraction, multiplication). The core idea is to check if the complement (target - current, target / current, etc.) has been seen before.

Two Sum (The Classic Example)

def two_sum(nums, target):
    """
    Find two numbers that sum to target
    Time: O(n), Space: O(n)
    """
    seen = {}  # number -> index
    
    for i, num in enumerate(nums):
        complement = target - num
        
        if complement in seen:
            return [seen[complement], i]
        
        seen[num] = i
    
    return None  # No solution found

# Variations

# Return values instead of indices
def two_sum_values(nums, target):
    """
    Return the actual values instead of indices
    """
    seen = set()
    
    for num in nums:
        complement = target - num
        
        if complement in seen:
            return [complement, num]
        
        seen.add(num)
    
    return None

# Count all pairs (not just return one)
def two_sum_count_pairs(nums, target):
    """
    Count all pairs that sum to target
    Time: O(n), Space: O(n)
    """
    freq = Counter(nums)
    count = 0
    
    for num in freq:
        complement = target - num
        
        if complement in freq:
            if complement == num:
                # Same number, choose 2 from frequency
                count += freq[num] * (freq[num] - 1) // 2
            elif num < complement:  # Avoid double counting
                count += freq[num] * freq[complement]
    
    return count

# Example
nums = [2, 7, 11, 15]
target = 9
print(two_sum(nums, target))  # [0, 1] (indices of 2 and 7)

Complement Variations

1. Two Sum II - Input Array is Sorted

def two_sum_sorted(numbers, target):
    """
    Two sum when array is sorted
    Can use two pointers instead of hash table
    Time: O(n), Space: O(1)
    """
    left, right = 0, len(numbers) - 1
    
    while left < right:
        current_sum = numbers[left] + numbers[right]
        
        if current_sum == target:
            return [left + 1, right + 1]  # 1-indexed as per problem
        elif current_sum < target:
            left += 1
        else:
            right -= 1
    
    return None

# Example
numbers = [2, 7, 11, 15]
target = 9
print(two_sum_sorted(numbers, target))  # [1, 2]

2. 3Sum

def three_sum(nums):
    """
    Find all triplets that sum to 0
    Time: O(n^2), Space: O(n)
    """
    nums.sort()  # O(n log n)
    result = []
    
    for i in range(len(nums) - 2):
        # Skip duplicates
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        
        left, right = i + 1, len(nums) - 1
        
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            
            if total == 0:
                result.append([nums[i], nums[left], nums[right]])
                
                # Skip duplicates
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                
                left += 1
                right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1
    
    return result

# Example
nums = [-1, 0, 1, 2, -1, -4]
print(three_sum(nums))
# [[-1, -1, 2], [-1, 0, 1]]

3. 4Sum

def four_sum(nums, target):
    """
    Find all quadruplets that sum to target
    Time: O(n^3), Space: O(n)
    """
    nums.sort()
    n = len(nums)
    result = []
    
    for i in range(n - 3):
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        
        for j in range(i + 1, n - 2):
            if j > i + 1 and nums[j] == nums[j - 1]:
                continue
            
            left, right = j + 1, n - 1
            
            while left < right:
                total = nums[i] + nums[j] + nums[left] + nums[right]
                
                if total == target:
                    result.append([nums[i], nums[j], nums[left], nums[right]])
                    
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    
                    left += 1
                    right -= 1
                elif total < target:
                    left += 1
                else:
                    right -= 1
    
    return result

# Example
nums = [1, 0, -1, 0, -2, 2]
target = 0
print(four_sum(nums, target))
# [[-2, -1, 1, 2], [-2, 0, 0, 2], [-1, 0, 0, 1]]

def has_pair_with_sum(arr, target):

Beyond Addition: Other Operations

5. Two Difference

def two_difference(nums, k):
    """
    Find pairs with difference k
    Time: O(n), Space: O(n)
    """
    seen = set()
    pairs = []
    
    for num in nums:
        if num - k in seen:
            pairs.append([num - k, num])
        if num + k in seen:
            pairs.append([num, num + k])
        seen.add(num)
    
    return pairs

# Example
nums = [1, 7, 5, 9, 2, 12, 3]
k = 2
print(two_difference(nums, k))  # [[5, 7], [1, 3]]

6. Pairs Equal to Product

def pairs_with_product(nums, target):
    """
    Find pairs whose product equals target
    Time: O(n), Space: O(n)
    """
    seen = {}
    pairs = []
    
    for num in nums:
        if num != 0 and target % num == 0:
            complement = target // num
            if complement in seen:
                pairs.append([complement, num])
        seen[num] = True
    
    return pairs

# Example
nums = [1, 2, 3, 4, 6]
target = 12
print(pairs_with_product(nums, target))  # [[2, 6], [3, 4]]
TIP: The complements pattern works for many problems involving pairs. Key insight: for two elements, we only need to know if we've seen the complement before processing the current element.

Grouping Problems

Organizing elements based on shared properties

Understanding Grouping Pattern

Grouping problems require organizing elements into categories based on shared properties. The hash table pattern maps a key (derived from each element) to a list of elements that share that key.

Group Anagrams

from collections import defaultdict

def group_anagrams(words):
    """
    Group words that are anagrams of each other
    Key: sorted characters of each word
    Time: O(n * k log k) where n = number of words, k = max word length
    Space: O(n * k)
    """
    groups = defaultdict(list)
    
    for word in words:
        # Sort characters to create key
        key = ''.join(sorted(word))
        groups[key].append(word)
    
    return list(groups.values())

# Alternative: Use frequency count as key (faster for long strings)
def group_anagrams_frequency(words):
    """
    Use character frequency as key
    Time: O(n * k * 26) = O(n * k)
    Space: O(n * k)
    """
    def get_key(word):
        freq = [0] * 26
        for char in word:
            freq[ord(char) - ord('a')] += 1
        return tuple(freq)
    
    groups = defaultdict(list)
    
    for word in words:
        key = get_key(word)
        groups[key].append(word)
    
    return list(groups.values())

# Example
words = ["eat", "tea", "tan", "ate", "nat", "bat"]
print(group_anagrams(words))
# [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]

Common Grouping Problems

1. Group Shifted Strings

def group_shifted_strings(strings):
    """
    Group strings that can be shifted to form each other
    Example: "abc" and "bcd" are in same group (shift all by 1)
    Time: O(n * k) where n = strings, k = max length
    Space: O(n * k)
    """
    def get_key(s):
        if not s:
            return ""
        
        # Key is the sequence of differences between consecutive characters
        key = []
        for i in range(1, len(s)):
            diff = (ord(s[i]) - ord(s[i - 1])) % 26
            key.append(str(diff))
        
        return ','.join(key)
    
    groups = defaultdict(list)
    
    for s in strings:
        groups[get_key(s)].append(s)
    
    return list(groups.values())

# Example
strings = ["abc", "bcd", "acef", "xyz", "az", "ba", "a", "z"]
print(group_shifted_strings(strings))
# [['abc', 'bcd', 'xyz'], ['az', 'ba'], ['acef'], ['a', 'z']]

2. Group Strings by Same Pattern

def group_by_pattern(strings):
    """
    Group strings that have the same character pattern
    Example: "abb" and "cdd" have same pattern (first unique, then duplicate)
    Time: O(n * k)
    """
    def get_pattern(s):
        mapping = {}
        pattern = []
        current = ord('a')
        
        for char in s:
            if char not in mapping:
                mapping[char] = chr(current)
                current += 1
            pattern.append(mapping[char])
        
        return ''.join(pattern)
    
    groups = defaultdict(list)
    
    for s in strings:
        pattern = get_pattern(s)
        groups[pattern].append(s)
    
    return list(groups.values())

# Example
strings = ["abb", "cdd", "foo", "xyz"]
print(group_by_pattern(strings))
# [['abb', 'cdd', 'foo'], ['xyz']]

3. Partition Labels

def partition_labels(s):
    """
    Partition string into as many parts as possible
    Each letter appears in at most one part
    Time: O(n), Space: O(1) - fixed alphabet size
    """
    # Last occurrence of each character
    last_occurrence = {}
    
    for i, char in enumerate(s):
        last_occurrence[char] = i
    
    result = []
    start = 0
    end = 0
    
    for i, char in enumerate(s):
        end = max(end, last_occurrence[char])
        
        if i == end:
            result.append(end - start + 1)
            start = i + 1
    
    return result

# Example
s = "ababcbacadefegdehijhklij"
print(partition_labels(s))  # [9, 7, 8]
# "ababcbaca", "defegde", "hijhklij"

4. Top K Frequent Words

from collections import Counter
import heapq

def top_k_frequent_words(words, k):
    """
    Return k most frequent words, sorted by frequency then lexicographically
    Time: O(n log k)
    Space: O(n)
    """
    freq = Counter(words)
    
    # Use min-heap with custom ordering
    heap = [(-count, word) for word, count in freq.items()]
    heapq.heapify(heap)
    
    result = []
    for _ in range(k):
        count, word = heapq.heappop(heap)
        result.append(word)
    
    return result

# Example
words = ["i", "love", "leetcode", "i", "love", "coding"]
k = 2
print(top_k_frequent_words(words, k))  # ["i", "love"]

Subarray Problems

Using prefix sums with hash tables

Subarray Sum with Hash Tables

Subarray problems can often be solved efficiently by combining prefix sums with hash tables. The key insight is that a subarray sum from index i to j equals prefix_sum[j+1] - prefix_sum[i].

Subarray Sum Equals K

def subarray_sum(nums, k):
    """
    Count number of subarrays that sum to k
    Time: O(n), Space: O(n)
    """
    count = 0
    prefix_sum = 0
    sum_count = {0: 1}  # prefix_sum -> count of occurrences
    
    for num in nums:
        prefix_sum += num
        
        # Check if (prefix_sum - k) exists
        if (prefix_sum - k) in sum_count:
            count += sum_count[prefix_sum - k]
        
        # Update count of current prefix_sum
        sum_count[prefix_sum] = sum_count.get(prefix_sum, 0) + 1
    
    return count

# Explanation:
# If prefix_sum[j] - prefix_sum[i] = k
# Then nums[i..j-1] sums to k
# We count how many times (prefix_sum - k) has occurred before

# Example
nums = [1, 1, 1]
k = 2
print(subarray_sum(nums, k))  # 2
# Subarrays: [1,1] at indices [0,1] and [1,2]

Subarray Variations

1. Maximum Size Subarray Sum Equals K

def max_size_subarray_sum(nums, k):
    """
    Find longest subarray that sums to k
    Time: O(n), Space: O(n)
    """
    prefix_sum = 0
    max_length = 0
    sum_to_index = {0: -1}
    
    for i, num in enumerate(nums):
        prefix_sum += num
        
        if prefix_sum - k in sum_to_index:
            length = i - sum_to_index[prefix_sum - k]
            max_length = max(max_length, length)
        
        if prefix_sum not in sum_to_index:
            sum_to_index[prefix_sum] = i
    
    return max_length

# Example
nums = [1, -1, 5, -2, 3]
k = 3
print(max_size_subarray_sum(nums, k))  # 4
# Longest subarray: [1, -1, 5, -2]

2. Subarray Sum Divisible by K

def subarrays_div_by_k(nums, k):
    """
    Count subarrays with sum divisible by k
    Time: O(n), Space: O(k)
    Uses modular arithmetic
    """
    count = 0
    prefix_sum = 0
    mod_count = {0: 1}
    
    for num in nums:
        prefix_sum += num
        mod = prefix_sum % k
        
        # Handle negative mod
        if mod < 0:
            mod += k
        
        if mod in mod_count:
            count += mod_count[mod]
        else:
            mod_count[mod] = 0
        
        mod_count[mod] += 1
    
    return count

# Example
nums = [4, 5, 0, -2, -3, 1]
k = 5
print(subarrays_div_by_k(nums, k))  # 7

3. Continuous Subarray Sum

def check_subarray_sum(nums, k):
    """
    Check if there's a subarray of size >= 2 that sums to multiple of k
    Time: O(n), Space: O(min(n, k))
    """
    if k == 0:
        # Check for consecutive zeros
        for i in range(len(nums) - 1):
            if nums[i] == 0 and nums[i + 1] == 0:
                return True
        return False
    
    prefix_sum = 0
    mod_to_index = {0: -1}
    
    for i, num in enumerate(nums):
        prefix_sum += num
        mod = prefix_sum % k
        
        if mod in mod_to_index:
            if i - mod_to_index[mod] >= 2:
                return True
        else:
            mod_to_index[mod] = i
    
    return False

# Example
nums = [23, 2, 4, 6, 7]
k = 6
print(check_subarray_sum(nums, k))  # True
# Subarray [2, 4] sums to 6

def find_subarray_with_sum(nums, target):
    """
    Find indices of subarray that sums to target (for positive numbers only)
    Time: O(n), Space: O(n)
    """
    prefix_sum = 0
    sum_to_index = {0: -1}
    
    for i, num in enumerate(nums):
        prefix_sum += num
        
        if prefix_sum - target in sum_to_index:
            start = sum_to_index[prefix_sum - target] + 1
            return [start, i]
        
        sum_to_index[prefix_sum] = i
    
    return None

Contiguous Array Problems

5. Contiguous Array (0s and 1s with equal count)

def find_max_length(nums):
    """
    Find longest contiguous array with equal number of 0 and 1
    Replace 0 with -1, find subarray with sum 0
    Time: O(n), Space: O(n)
    """
    count = 0
    max_length = 0
    count_to_index = {0: -1}
    
    for i, num in enumerate(nums):
        # Treat 0 as -1
        count += 1 if num == 1 else -1
        
        if count in count_to_index:
            max_length = max(max_length, i - count_to_index[count])
        else:
            count_to_index[count] = i
    
    return max_length

# Example
nums = [0, 1, 0]
print(find_max_length(nums))  # 2
# Longest: [0, 1] or [1, 0]

6. Binary Subarrays with Sum

def num_subarrays_with_sum(nums, goal):
    """
    Count binary subarrays with sum equal to goal
    Time: O(n), Space: O(n)
    """
    from collections import defaultdict
    
    prefix_sum = 0
    count = 0
    sum_count = defaultdict(int)
    sum_count[0] = 1
    
    for num in nums:
        prefix_sum += num
        
        if prefix_sum - goal in sum_count:
            count += sum_count[prefix_sum - goal]
        
        sum_count[prefix_sum] += 1
    
    return count

# Example
nums = [1, 0, 1, 0, 1]
goal = 2
print(num_subarrays_with_sum(nums, goal))  # 4
KEY INSIGHT: For subarray problems, prefix_sum[i] - prefix_sum[j] gives sum of elements from index j to i-1. Use hash table to store (prefix_sum -> index) or (prefix_sum -> count) to efficiently find subarrays meeting criteria.

1. Choose the Right Data Structure

for char in string:

8. Early Termination

def check_anagram(s1, s2):
    """
    Check if two strings are anagrams with early termination
    """
    if len(s1) != len(s2):
        return False
    
    # Quick termination for very different strings
    if sorted(s1) == sorted(s2):
        return True
    
    # Use frequency counts
    freq = [0] * 26
    
    for c1, c2 in zip(s1, s2):
        idx1 = ord(c1) - ord('a')
        idx2 = ord(c2) - ord('a')
        
        freq[idx1] += 1
        freq[idx2] -= 1
        
        # Early check: if any count exceeds string length
        if abs(freq[idx1]) > len(s1) or abs(freq[idx2]) > len(s1):
            return False
    
    return all(f == 0 for f in freq)
