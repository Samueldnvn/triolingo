Understanding the Sliding Window

The sliding window technique is a method for processing subarrays or substrings efficiently. Instead of recomputing everything for each position, you "slide" a window across the data, adjusting the window incrementally as you move. This typically reduces complexity from O(n^2) or O(n*k) to O(n).

CORE IDEA

For a window of size k covering indices [left, right]:
- Move right forward one step: window becomes [left+1, right+1]
- Remove element at left, add element at right+1
- This update is O(1) instead of O(k) for recomputing

When to Use Sliding Window

Pattern Use Sliding Window? Alternative
Find subarray of fixed size k with max sum YES Variations: prefix sum
Find longest substring with only k distinct chars YES (variable) Binary search + hash
Find subarrays with sum EXACTLY K NO Hash table + prefix sum
Contains elements with constraints YES (variable) Two pointers
Need to backtrack to previous positions NO Backtracking/DFS
Maximum sum of ANY subarray (no size constraint) NO Kadane's algorithm

Time Complexity Analysis

Sliding window achieves O(n) complexity because each element is processed at most twice: once when the window expands and once when it contracts. This is the hallmark of two-pointer techniques.

KEY INSIGHT: The power of sliding window comes from incremental updates. You never recompute from scratch - you adjust based on what's leaving and entering the window.

Fixed-Size Sliding Window

Windows with constant width k

Understanding Fixed-Size Windows

Fixed-size windows maintain a constant width k as they slide through the array or string. They're useful when you need to analyze or compute something about all subarrays/strings of a specific length.

Template 1: Sum/Aggregation

def fixed_window_sum(arr, k):
    """
    Find sum of each window of size k
    Time: O(n), Space: O(1) (excluding output)
    """
    n = len(arr)
    if n < k:
        return None
    
    # Compute sum of first window
    window_sum = sum(arr[:k])
    result = [window_sum]
    
    # Slide window, adding new element, removing old
    for i in range(k, n):
        window_sum = window_sum - arr[i - k] + arr[i]
        result.append(window_sum)
    
    return result


# Example
arr = [2, 1, 5, 1, 3, 2]
k = 3
print(fixed_window_sum(arr, k))
# [8, 7, 9, 6]
# Windows: [2,1,5]=8, [1,5,1]=7, [5,1,3]=9, [1,3,2]=6

1. Maximum Sum Subarray of Size K

def max_sum_subarray(arr, k):
    """
    Find maximum sum of any subarray of size k
    Time: O(n), Space: O(1)
    """
    n = len(arr)
    if n < k:
        return None
    
    # First window
    window_sum = sum(arr[:k])
    max_sum = window_sum
    
    # Slide window
    for i in range(k, n):
        window_sum = window_sum - arr[i - k] + arr[i]
        max_sum = max(max_sum, window_sum)
    
    return max_sum


# With indices
def max_sum_subarray_with_indices(arr, k):
    """
    Return max sum and the start index
    """
    n = len(arr)
    if n < k:
        return None, None
    
    window_sum = sum(arr[:k])
    max_sum = window_sum
    max_start = 0
    
    for i in range(k, n):
        window_sum = window_sum - arr[i - k] + arr[i]
        if window_sum > max_sum:
            max_sum = window_sum
            max_start = i - k + 1
    
    return max_sum, max_start

2. Average of All Subarrays of Size K

def average_of_subarrays(arr, k):
    """
    Return list of averages for each window of size k
    Time: O(n), Space: O(n) for result
    """
    n = len(arr)
    result = []
    window_sum = 0.0
    window_start = 0
    
    for window_end in range(n):
        window_sum += arr[window_end]
        
        # Window is size k
        if window_end >= k - 1:
            result.append(window_sum / k)
            window_sum -= arr[window_start]
            window_start += 1
    
    return result

# Example
arr = [1, 3, 2, 6, -1, 4, 1, 8, 2]
k = 5
print(average_of_subarrays(arr, k))
# [2.2, 2.8, 2.4, 3.6, 2.8]

3. Maximum of All Subarrays of Size K

from collections import deque

def max_of_subarrays(arr, k):
    """
    Find maximum in each subarray of size k
    Uses deque to maintain candidates in descending order
    Time: O(n), Space: O(k)
    """
    n = len(arr)
    if n < k:
        return []
    
    deque_idx = deque()  # Stores indices, values in descending order
    result = []
    
    for i in range(n):
        # Remove indices not in window
        while deque_idx and deque_idx[0] <= i - k:
            deque_idx.popleft()
        
        # Remove smaller elements from back
        while deque_idx and arr[deque_idx[-1]] < arr[i]:
            deque_idx.pop()
        
        deque_idx.append(i)
        
        # Window is full, add to result
        if i >= k - 1:
            result.append(arr[deque_idx[0]])
    
    return result

# Example
arr = [1, 3, -1, -3, 5, 3, 6, 7]
k = 3
print(max_of_subarrays(arr, k))
# [3, 3, 5, 5, 6, 7]

4. Count Distinct Elements in Each Window

from collections import defaultdict

def count_distinct_in_windows(arr, k):
    """
    Count distinct elements in each window of size k
    Time: O(n), Space: O(k)
    """
    n = len(arr)
    if n < k:
        return []
    
    freq_map = defaultdict(int)
    distinct_count = 0
    result = []
    
    # First window
    for i in range(k):
        if freq_map[arr[i]] == 0:
            distinct_count += 1
        freq_map[arr[i]] += 1
    result.append(distinct_count)
    
    # Slide window
    for i in range(k, n):
        # Remove element leaving window
        element_out = arr[i - k]
        freq_map[element_out] -= 1
        if freq_map[element_out] == 0:
            distinct_count -= 1
        
        # Add element entering window
        element_in = arr[i]
        if freq_map[element_in] == 0:
            distinct_count += 1
        freq_map[element_in] += 1
        
        result.append(distinct_count)
    
    return result

# Example
arr = [1, 2, 1, 3, 4, 2, 3]
k = 4
print(count_distinct_in_windows(arr, k))
# [3, 4, 4, 3]

5. Anagrams in String (Fixed Window)

from collections import Counter

def find_anagrams(s, p):
    """
    Find all start indices of p's anagrams in s
    Uses sliding window with character counts
    Time: O(n), Space: O(1) - fixed alphabet
    """
    n, m = len(s), len(p)
    if n < m:
        return []
    
    result = []
    p_count = Counter(p)
    window_count = Counter(s[:m])
    
    if window_count == p_count:
        result.append(0)
    
    # Slide window
    for i in range(m, n):
        # Add new character
        window_count[s[i]] += 1
        
        # Remove old character
        char_out = s[i - m]
        window_count[char_out] -= 1
        if window_count[char_out] == 0:
            del window_count[char_out]
        
        # Check match
        if window_count == p_count:
            result.append(i - m + 1)
    
    return result

# Example
s = "cbaebabacd"
p = "abc"
print(find_anagrams(s, p))  # [0, 6]
# Substrings: "cba" at 0, "bac" at 6
TIP: For fixed-size windows, compute the first window, then update incrementally. The update formula is always: new_value = old_value - element_leaving + element_entering

Variable-Size Sliding Window

Windows that grow and shrink based on constraints

Understanding Variable-Size Windows

Variable-size windows adjust their width dynamically based on problem constraints. The window grows as long as the constraint is satisfied and shrinks when violated. Common for problems like "longest substring with at most K distinct characters."

Template 2: Expand Until Constraint, Shrink When Violated

def variable_window_template(arr, constraint_func):
    """
    Generic template for variable-size window
    constraint_func(window): bool - returns True if constraint met
    """
    left = 0
    result = []
    
    for right in range(len(arr)):
        # Expand window from right
        right_expand(right)
        
        # Shrink from left while constraint violated
        while left <= right and not constraint_func(window):
            left_shrink(left)
            left += 1
        
        # Window now satisfies constraint, process result
        update_result()
    
    return result

Classic Variable-Size Problems

1. Longest Substring with At Most K Distinct Characters

def longest_substring_k_distinct(s, k):
    """
    Find longest substring with at most K distinct characters
    Time: O(n), Space: O(k)
    """
    n = len(s)
    if k == 0 or n == 0:
        return 0
    
    left = 0
    char_freq = {}
    max_length = 0
    
    for right in range(n):
        # Add character to window
        char_freq[s[right]] = char_freq.get(s[right], 0) + 1
        
        # Shrink window if more than K distinct chars
        while len(char_freq) > k:
            left_char = s[left]
            char_freq[left_char] -= 1
            if char_freq[left_char] == 0:
                del char_freq[left_char]
            left += 1
        
        # Update max length
        max_length = max(max_length, right - left + 1)
    
    return max_length

# Example
s = "araaci"
k = 2
print(longest_substring_k_distinct(s, k))  # 4
# Longest: "araa"

2. Longest Substring Without Repeating Characters

def length_of_longest_substring(s):
    """
    Find longest substring without repeating characters
    Time: O(n), Space: O(min(n, charset))
    """
    char_index = {}  # character -> last index seen
    left = 0
    max_length = 0
    
    for right, char in enumerate(s):
        # If character seen and is in current window
        if char in char_index and char_index[char] >= left:
            left = char_index[char] + 1
        
        char_index[char] = right
        max_length = max(max_length, right - left + 1)
    
    return max_length

# Example
print(length_of_longest_substring("abcabcbb"))  # 3 ("abc")
print(length_of_longest_substring("bbbbb"))     # 1 ("b")
print(length_of_longest_substring("pwwkew"))    # 3 ("wke")

3. Longest Substring with Same Letters After Replacement

def character_replacement(s, k):
    """
    Longest substring with same letters after at most K replacements
    Time: O(n), Space: O(1) - 26 letters only
    """
    n = len(s)
    char_count = [0] * 26
    max_count = 0
    max_length = 0
    window_start = 0
    
    for window_end in range(n):
        idx = ord(s[window_end]) - ord('A')
        char_count[idx] += 1
        max_count = max(max_count, char_count[idx])
        
        # Shrink window if more than K replacements needed
        while (window_end - window_start + 1) - max_count > k:
            left_idx = ord(s[window_start]) - ord('A')
            char_count[left_idx] -= 1
            window_start += 1
        
        max_length = max(max_length, window_end - window_start + 1)
    
    return max_length

# Example
s = "ABAB"
k = 2
print(character_replacement(s, k))  # 4
# Replace 2 As to 2 Bs or 2 Bs to 2 As

4. Longest Subarray with Ones After Replacement

def longest_ones(arr, k):
    """
    Longest subarray containing only 1s after flipping at most K 0s
    Time: O(n), Space: O(1)
    """
    left = 0
    max_length = 0
    zero_count = 0
    
    for right in range(len(arr)):
        if arr[right] == 0:
            zero_count += 1
        
        # Shrink window if more than K zeros
        while zero_count > k:
            if arr[left] == 0:
                zero_count -= 1
            left += 1
        
        max_length = max(max_length, right - left + 1)
    
    return max_length

# Example
arr = [0, 1, 1, 0, 0, 0, 1, 1, 0, 1, 1]
k = 2
print(longest_ones(arr, k))  # 6

5. Permutation in String

def check_inclusion(s1, s2):
    """
    Check if s2 contains a permutation of s1
    Time: O(n + m), Space: O(1)
    """
    n, m = len(s1), len(s2)
    if n > m:
        return False
    
    s1_count = [0] * 26
    s2_count = [0] * 26
    
    # Initialize first window
    for i in range(n):
        s1_count[ord(s1[i]) - ord('a')] += 1
        s2_count[ord(s2[i]) - ord('a')] += 1
    
    matches = 0
    for i in range(26):
        if s1_count[i] == s2_count[i]:
            matches += 1
    
    # Slide window
    left = 0
    for right in range(n, m):
        if matches == 26:
            return True
        
        # Add new character
        idx = ord(s2[right]) - ord('a')
        if s2_count[idx] == s1_count[idx]:
            matches -= 1
        s2_count[idx] += 1
        if s2_count[idx] == s1_count[idx]:
            matches += 1
        
        # Remove old character
        idx = ord(s2[left]) - ord('a')
        if s2_count[idx] == s1_count[idx]:
            matches -= 1
        s2_count[idx] -= 1
        if s2_count[idx] == s1_count[idx]:
            matches += 1
        
        left += 1
    
    return matches == 26

# Example
s1 = "ab"
s2 = "eidbaooo"
print(check_inclusion(s1, s2))  # True

Template 1: Fixed Size - Maximum/Minimum

def fixed_window_max_min(arr, k):
    """
    Find max/min in each window of fixed size k
    """
    n = len(arr)
    if n < k:
        return None
    
    # Initialize
    current = compute_function(arr[:k])
    result = [current]
    
    # Slide window
    for i in range(k, n):
        # Update: remove left, add right
        current = update_function(current, arr[i-k], arr[i])
        result.append(current)
    
    return result

# Example: maximum
def fixed_window_max(arr, k):
    result = []
    from collections import deque
    dq = deque()
    
    for i, num in enumerate(arr):
        # Remove out-of-window elements
        while dq and dq[0] <= i - k:
            dq.popleft()
        
        # Maintain decreasing order
        while dq and arr[dq[-1]] < num:
            dq.pop()
        
        dq.append(i)
        
        if i >= k - 1:
            result.append(arr[dq[0]])
    
    return result

Template 4: Find Minimum/Maximum Window

def find_max_window_template(arr, constraint_func):
    """
    Find maximum window satisfying constraint
    Returns (window_start, window_end, window_length)
    """
    left = 0
    max_length = 0
    result = (0, 0, 0)
    
    for right in range(len(arr)):
        # Update window state with arr[right]
        
        # Shrink while constraint violated
        while left <= right and not constraint_func(window):
            # Update window state removing arr[left]
            left += 1
        
        # Window valid, check if maximum
        if constraint_func(window):
            if right - left + 1 > max_length:
                max_length = right - left + 1
                result = (left, right, max_length)
    
    return result

# Example: longest valid parentheses substring
def longest_valid_parentheses(s):
    left = 0
    max_len = 0
    stack = [-1]
    
    for i, char in enumerate(s):
        if char == '(':
            stack.append(i)
        else:
            stack.pop()
            if not stack:
                stack.append(i)
            else:
                max_len = max(max_len, i - stack[-1])
    
    return max_len

Common Problems

Frequently encountered sliding window scenarios

1. Minimum Window Substring

def min_window(s, t):
    """
    Find minimum window in s containing all characters of t
    Time: O(n + m), Space: O(k) where k = distinct chars in t
    """
    from collections import Counter
    
    if not s or not t or len(t) > len(s):
        return ""
    
    t_count = Counter(t)
    required = len(t_count)
    
    left = 0
    formed = 0
    window_counts = {}
    
    result = (float('inf'), None, None)  # (length, left, right)
    
    for right, char in enumerate(s):
        window_counts[char] = window_counts.get(char, 0) + 1
        
        if char in t_count and window_counts[char] == t_count[char]:
            formed += 1
        
        # Try to shrink window from left
        while left <= right and formed == required:
            char = s[left]
            
            # Update最小窗口
            if right - left + 1 < result[0]:
                result = (right - left + 1, left, right)
            
            # Remove left character
            window_counts[char] -= 1
            if char in t_count and window_counts[char] < t_count[char]:
                formed -= 1
            
            left += 1
    
    return "" if result[0] == float('inf') else s[result[1]:result[2]+1]

# Example
s = "ADOBECODEBANC"
t = "ABC"
print(min_window(s, t))  # "BANC"

2. Minimum Size Subarray Sum

def min_subarray_sum(nums, target):
    """
    Find minimal length subarray with sum >= target
    Time: O(n), Space: O(1)
    """
    n = len(nums)
    min_length = float('inf')
    window_sum = 0
    left = 0
    
    for right in range(n):
        window_sum += nums[right]
        
        # Shrink window while sum >= target
        while window_sum >= target:
            min_length = min(min_length, right - left + 1)
            window_sum -= nums[left]
            left += 1
    
    return 0 if min_length == float('inf') else min_length

# Example
nums = [2, 3, 1, 2, 4, 3]
target = 7
print(min_subarray_sum(nums, target))  # 2
# Subarray [4, 3]

def total_fruit(tree):
    """
    Longest contiguous subarray with at most 2 distinct elements
    Same as "longest substring with at most 2 distinct chars"
    Time: O(n), Space: O(1)
    """
    left = 0
    fruit_count = {}
    max_fruits = 0
    
    for right, fruit in enumerate(tree):
        fruit_count[fruit] = fruit_count.get(fruit, 0) + 1
        
        while len(fruit_count) > 2:
            fruit_count[tree[left]] -= 1
            if fruit_count[tree[left]] == 0:
                del fruit_count[tree[left]]
            left += 1
        
        max_fruits = max(max_fruits, right - left + 1)
    
    return max_fruits

4. String Concatenation

def find_all_substrings(s, words):
    """
    Find all starting indices of substrings that are concatenation of words
    Words all have same length
    Time: O(n * m * k) where n = len(s), m = words count, k = word length
    """
    if not s or not words:
        return []
    
    word_length = len(words[0])
    total_length = word_length * len(words)
    result = []
    word_count = {}
    
    for word in words:
        word_count[word] = word_count.get(word, 0) + 1
    
    # Try each starting position
    for i in range(len(s) - total_length + 1):
        seen = {}
        j = 0
        
        # Check each word in the window
        while j < len(words):
            start = i + j * word_length
            word = s[start:start + word_length]
            
            if word not in word_count:
                break
            
            seen[word] = seen.get(word, 0) + 1
            
            if seen[word] > word_count[word]:
                break
            
            j += 1
        
        if j == len(words):
            result.append(i)
    
    return result

# Example
s = "wordgoodgoodgoodbestword"
words = ["word","good","best","word"]
print(find_all_substrings(s, words))  # []

def find_repeated_dna_sequences(s):
    """
    Find all 10-letter sequences occurring more than once
    Time: O(n), Space: O(n)
    """
    seen = {}
    repeated = []
    
    for i in range(len(s) - 9):
        sequence = s[i:i+10]
        
        if sequence in seen:
            seen[sequence] += 1
            if seen[sequence] == 2:
                repeated.append(sequence)
        else:
            seen[sequence] = 1
    
    return repeated

# ["AAAAACCCCC", "CCCCCAAAAA"]

1. Find a subarray/substring satisfying some condition

Optimization Techniques

Performance tuning for sliding window solutions

1. Choose Data Structures Wisely

# For tracking distinct items in window
# Slow: Check entire window each time
while violates_constraint_by_scanning(window):
    ...

# Fast: Track with hash map
while len(visited) > k:
    ...

# For maximum/minimum in window
# Slow: Recompute each time
max_val = max(window[left:right+1])

# Fast: Use deque or heap
while dq and arr[dq[-1]] < arr[i]:
    dq.pop()

2. Avoid Unnecessary Operations

# Slow: Update metric after every slide
for i in range(k, n):
    slide_window()
    metric = compute_metric(window)  # O(k) each time

# Fast: Update metric incrementally
metric = compute_metric(first_window)  # O(k) once
for i in range(k, n):
    metric = metric - arr[i-k] + arr[i]  # O(1) each time
