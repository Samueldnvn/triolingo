IoT Hardware Fundamentals

Sensors, Microcontrollers, and Hardware Interface Programming

This module covers the hardware foundation of IoT systems, including sensor types, microcontroller architectures, communication protocols, and programming techniques for interfacing Java with physical hardware.

KEY HARDWARE COMPONENTS
  • Microcontrollers: Small computers on single integrated circuits
  • Single Board Computers: Full computers like Raspberry Pi, BeagleBone
  • Sensors: Devices that detect and measure physical phenomena
  • Actuators: Devices that perform physical actions
  • Communication Modules: WiFi, Bluetooth, and other network interfaces

SENSOR TYPES AND CHARACTERISTICS

Digital vs Analog Sensors

Sensors can be broadly classified as digital or analog, based on their output signal type:

Aspect Digital Sensors Analog Sensors
Output Discrete values (0/1 or multiple digital values) Continuous voltage level
Interface Digital I/O, SPI, I2C ADC required
Resolution Fixed by sensor design Depends on ADC resolution
Noise Immunity High Lower, susceptible to noise
Examples Digital temperature, motion sensors Thermistor, photoresistor

Common Sensor Categories

ENVIRONMENTAL SENSORS
  • Temperature: Measures thermal energy (DS18B20, DHT22, TMP36)
  • Humidity: Measures moisture in air (DHT11/22, BME280)
  • Pressure: Measures barometric pressure (BMP280, MPL3115)
  • Air Quality: Detects gases and particles (MQ-135, PMS5003)
MOTION AND POSITION SENSORS
  • Accelerometer: Measures acceleration force (MPU6050)
  • Gyroscope: Measures angular velocity (MPU6050)
  • Magnetometer: Measures magnetic field (HMC5883L)
  • PIR Motion: Detects infrared motion (HC-SR501)
  • Ultrasonic: Distance measurement (HC-SR04)
LIGHT AND OPTICAL SENSORS
  • Photoresistor: Light-dependent resistor (LDR)
  • Photodiode: Converts light to current
  • RGB Color: Detects color values (TCS3200)
  • Light Intensity: Measures lux (BH1750, TSL2561)

SENSOR PROGRAMMING WITH JAVA

Reading Digital Sensors

Digital sensors provide discrete output values and can be read directly through GPIO or digital communication protocols.

// Reading a digital temperature sensor (DS18B20 via Pi4J)

import com.pi4j.io.gpio.*;

public class DigitalSensorReader {
  private final GpioPinDigitalInput pin;

  public DigitalSensorReader() {
    GpioController gpio = GpioFactory.getInstance();
    pin = gpio.provisionDigitalInputPin(RaspiPin.GPIO_04);
  }

  public boolean readState() {
    return pin.getState().isHigh();
  }
}

Sensor Polling Pattern

Most sensor applications use a polling pattern to periodically read sensor values.

// Continuous sensor reading with interval

public class SensorPoller {
  private final Sensor sensor;
  private final long intervalMs;

  public void start() {
    new Thread(() -> {
      while (true) {
        try {
          SensorData data = sensor.read();
          processData(data);
          Thread.sleep(intervalMs);
        } catch (InterruptedException e) {
          e.printStackTrace();
        }
      }
    }).start();
  }
}

Event-Driven Sensor Reading

For digital sensors like motion detectors or buttons, event-driven programming is often more efficient than polling.

// Event-driven sensor reading with Pi4J

public class EventSensorReader {
  public void setupMotionDetector() {
    GpioController gpio = GpioFactory.getInstance();
    GpioPinDigitalInput pir = gpio
      .provisionDigitalInputPin(RaspiPin.GPIO_05, PinPullResistance.PULL_DOWN);

    pir.addListener((GpioPinListenerDigital) event -> {
      if (event.getState().isHigh()) {
        handleMotionDetected();
      }
    });
  }
}

ANALOG-TO-DIGITAL CONVERSION

ADC Fundamentals

Analog-to-Digital Converters (ADC) are essential for reading analog sensors that output continuous voltage levels. The ADC converts these continuous signals into discrete digital values.

ADC Resolution and Digital Value
$$DigitalValue = \frac{Vin}{Vref} \times (2^n - 1)$$
Where:
Vin = Input voltage
Vref = Reference voltage
n = Number of bits
ADC CHARACTERISTICS
  • Resolution: Number of bits determines precision (8-bit = 256 values, 10-bit = 1024, 12-bit = 4096)
  • Reference Voltage: Maximum voltage that can be measured
  • Sampling Rate: How many samples per second can be taken
  • Channels: Number of analog inputs available

Reading Analog Sensors

// Reading analog sensor with Raspberry Pi ADC (MCP3008 via SPI)

public class AnalogSensorReader {
  public double readAnalog(int channel) {
    // SPI communication with MCP3008 ADC
    int rawValue = readMCP3008(channel);

    double voltage = (rawValue / 1023.0) * 3.3;
    return voltage;
  }
}
RASPBERRY PI DOES NOT HAVE BUILTIN ADC, SO EXTERNAL CHIPS LIKE MCP3008 ARE REQUIRED FOR ANALOG SENSOR READING
