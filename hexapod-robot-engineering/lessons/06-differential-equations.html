Module Overview

Differential equations model robot dynamics. From mass-spring-damper to state-space representation, they provide mathematical tools for analysis and design.

Second-Order ODE

Characteristic equation: \(ar^2 + br + c = 0\). Roots determine solution form. Underdamped: \(\zeta < 1\) oscillates with decaying amplitude \(e^{-\zeta\omega_nt}\).

Mass-Spring-Damper

\(m\ddot{x} + c\dot{x} + kx = F(t).\) Dividing: \(\ddot{x} + 2\zeta\omega_n\dot{x} + \omega_n^2 x = F(t)/m.\) Where \(\zeta = c/(2\sqrt{mk})\) is damping ratio.
