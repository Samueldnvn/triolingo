Matrix Representation

Every linear transformation between finite-dimensional vector spaces can be represented by a matrix. This matrix depends on the choice of basis for both the domain and codomain.

Standard Matrix of a Linear Transformation

If T is a linear transformation from R^n to R^m, the matrix A that represents T (called the standard matrix) is formed by taking T(e1), T(e2), ..., T(en) as columns, where e1, e2, ..., en are the standard basis vectors of R^n.

A = [T(e1) | T(e2) | ... | T(en)]

Then for any vector x in R^n, we have T(x) = Ax.

Example 1: Finding a Standard Matrix

Find the standard matrix A for the linear transformation T: R^2 -> R^2 that reflects vectors across the x-axis.

Step 1: Apply T to the standard basis vectors e1 = (1,0) and e2 = (0,1).

T(e1) = T(1,0) = (1,0)
T(e2) = T(0,1) = (0,-1)

Step 2: Form the matrix with these as columns.

A = \begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix}

Verification: If x = (3,4), then T(x) = (3,-4). Check using Ax:

Ax = \begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix} \begin{bmatrix} 3 \\ 4 \end{bmatrix} = \begin{bmatrix} 3 \\ -4 \end{bmatrix}
Example 2: Rotation Transformation

Find the standard matrix for a rotation by angle theta about the origin in R^2.

Step 1: Apply the rotation to the standard basis vectors.

T(e1) = (cos(theta), sin(theta))
T(e2) = (-sin(theta), cos(theta))

Step 2: Form the matrix.

R(theta) = \begin{bmatrix} cos(theta) & -sin(theta) \\ sin(theta) & cos(theta) \end{bmatrix}

This is the standard rotation matrix. For a 90-degree counterclockwise rotation:

R(90) = \begin{bmatrix} 0 & -1 \\ 1 & 0 \end{bmatrix}
Matrix Representation with Non-Standard Bases

To find the matrix representation of T with respect to non-standard bases B and B', we use the formula:

[T(x)]_B' = [T]_B^B' [x]_B

Where [T]_B^B' is the matrix representing T with respect to basis B (domain) and basis B' (codomain).

Note

To find [T]_B^B', apply T to each basis vector in B, express the result in terms of B', and use those coefficients as columns.

Theorem

If dim(V) = n and dim(W) = m, then the set of all linear transformations from V to W is isomorphic to the set of all m x n matrices.

Composition of Linear Transformations

If T1: V -> W and T2: W -> Z are linear transformations, then the composition T2(T1(v)) is also a linear transformation. The matrix representation is:

[T2(T1)] = [T2][T1]

Order matters. If [T1] is m x n and [T2] is p x m, then [T2(T1)] is p x n.

Properties of Matrix Representations
Key Properties
  • The matrix representation of a linear transformation is unique given fixed bases.
  • The matrix representation of T + S is [T] + [S].
  • The matrix representation of cT is c[T] for any scalar c.
  • The rank of the matrix equals the rank of the transformation.
  • Nullity is preserved: nullity([T]) = nullity(T).
Practice Problems
Problem 1: Find the standard matrix for the transformation T: R^2 -> R^2 that projects vectors onto the line y = x.
Problem 2: Find the standard matrix for scaling by a factor of 2 in both x and y directions.
Problem 3: If A = [[1,2], [3,4]] and B = [[0,1], [1,0]], find the matrix representing the transformation B after A (i.e., BA).
Problem 4: Find the standard matrix for a 60-degree rotation about the origin in R^2 (use cos(60) = 0.5, sin(60) = sqrt(3)/2).
Problem 5: What is the matrix representing a shear transformation in the x-direction by factor k?
