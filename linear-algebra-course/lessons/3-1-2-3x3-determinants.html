Cofactor Expansion

For 3x3 matrix, we use cofactor expansion along any row or column.

det([[a, b, c], [d, e, f], [g, h, i]]) = a(ei - fh) - b(di - fg) + c(dh - eg)
Example

Calculate det([[2, 0, 0], [1, 3, 0], [2, 1, 4]])

Expanding along first row:

det = 2 * det([[3, 0], [1, 4]]) - 0 + 0

det = 2 * (12 - 0) = 2 * 12 = 24

Solution: 24

Tip

Expand along the row or column with the most zeros to simplify calculations!
