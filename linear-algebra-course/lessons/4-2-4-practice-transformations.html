Module Quiz: Linear Transformations

Test your understanding of linear transformations, kernel, range, and matrix representations. This quiz covers all concepts from Module 4.2.

Linear Transformation Properties
Question 1: Which of the following is always true for a linear transformation T?



Answer: A
A linear transformation must satisfy T(0) = 0. This follows from linearity: T(0) = T(0+0) = T(0) + T(0), which implies T(0) = 0.
Question 2: If T: R^3 -> R^2 and T([1,0,0]) = [2,1], T([0,1,0]) = [0,3], T([0,0,1]) = [1,1], what is T([2,1,-1])?
Answer: [5,2]
First, find the matrix A = [[2,0,1], [1,3,1]]. Then T([2,1,-1]) = A*[2,1,-1] = [5,2].
Question 3: The kernel of a linear transformation T: V -> W consists of:



Answer: C
The kernel of T is the set of all vectors v in the domain V such that T(v) = 0 in the codomain W.
Question 4: If T: R^3 -> R^3 has matrix A = [[1,2,3], [0,1,1], [0,0,1]], what is the dimension of ker(T)?



Answer: A
The matrix A is upper-triangular with all diagonal entries equal to 1, so det(A) = 1. This means A is invertible, so the only solution to Ax = 0 is x = 0. Thus, ker(T) = {0} with dimension 0.
Matrix Representations
Question 5: Find the standard matrix for a 90-degree clockwise rotation in R^2.
Answer: [[0,1], [-1,0]]
Applying the 90-degree clockwise rotation to e1 = (1,0) gives (0,-1). Applying it to e2 = (0,1) gives (1,0). These form the columns.
Question 6: If T(x) = Ax where A = [[1,-1], [2,3]], find the matrix for T^2 (i.e., T(T(x))).
Answer: [[-1,-4], [8,7]]
The matrix for T^2 is A^2 = [[1,-1], [2,3]] * [[1,-1], [2,3]] = [[-1,-4], [8,7]].
Question 7: What is the rank of the transformation T: R^3 -> R^2 with matrix A = [[1,2,1], [2,4,2]]?



Answer: B
Row 2 is 2 times Row 1, so the row-rank is 1. This equals the column-rank, so rank(T) = 1. By the rank-nullity theorem, nullity = 3 - 1 = 2.
Advanced Problems
Question 8: If T: P_2 -> P_2 is defined by T(p(x)) = p'(x) (derivative), find the matrix of T with respect to basis {1, x, x^2}.
Answer: [[0,1,0], [0,0,2], [0,0,0]]
T(1) = 0 = 0*1 + 0*x + 0*x^2, T(x) = 1 = 1*1 + 0*x + 0*x^2, T(x^2) = 2x = 0*1 + 2*x + 0*x^2.
Question 9: Which of the following transformations is NOT linear?



Answer: B
T(x,y) = (x+1, y+1) is not linear because T(0,0) = (1,1) ≠ (0,0), which violates a required property of linear transformations.
Question 10: If T: R^2 -> R^2 has standard matrix A, what is the matrix for the transformation that reflects across the line y = 2x?
Answer: [[-0.6,0.8], [0.8,0.6]]
For reflection across line y = mx, the matrix is (1/(1+m^2)) * [[1-m^2, 2m], [2m, m^2-1]]. For m=2: [[-3/5, 4/5], [4/5, 3/5]].
