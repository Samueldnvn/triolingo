Gram-Schmidt Process

The Gram-Schmidt process is an algorithm for converting any basis of an inner product space into an orthonormal basis (a basis where all vectors are orthogonal and have unit length). It works by iteratively subtracting projections.

The Algorithm

Given a basis {v1, v2, ..., vn}, the Gram-Schmidt process produces an orthonormal basis {u1, u2, ..., un} as follows:

u_1 = v_1 / |v_1|

For each subsequent vector k = 2, 3, ..., n:

w_k = v_k - sum(v_k dot u_i * u_i, i=1 to k-1)
u_k = w_k / |w_k|
Key Idea

At each step, we subtract all projections of v_k onto the previously formed orthonormal vectors. What remains is orthogonal to all previous vectors.

Example 1: 2D Gram-Schmidt

Apply Gram-Schmidt to the basis {v1 = [1, 1], v2 = [1, 0]}.

Step 1: Create u1 (normalize v1)

|v1| = sqrt(1^2 + 1^2) = sqrt(2)
u1 = [1/sqrt(2), 1/sqrt(2)]

Step 2: Create w2 (remove projection onto u1)

v2 dot u1 = (1)(1/sqrt(2)) + (0)(1/sqrt(2)) = 1/sqrt(2)
w2 = v2 - (v2 dot u1) u1 = [1, 0] - (1/sqrt(2))[1/sqrt(2), 1/sqrt(2)]
w2 = [1, 0] - [1/2, 1/2] = [1/2, -1/2]

Step 3: Normalize w2 to get u2

|w2| = sqrt((1/2)^2 + (-1/2)^2) = sqrt(1/4 + 1/4) = sqrt(1/2) = 1/sqrt(2)
u2 = [1/2, -1/2] / (1/sqrt(2)) = [1/sqrt(2), -1/sqrt(2)]

Result: Orthonormal basis = {[1/sqrt(2), 1/sqrt(2)], [1/sqrt(2), -1/sqrt(2)]}

Example 2: 3D Gram-Schmidt

Apply Gram-Schmidt to {v1 = [1, 0, 0], v2 = [1, 1, 0], v3 = [1, 1, 1]}.

u1: Normalize v1 = [1, 0, 0]. u1 = [1, 0, 0].

For v2:

v2 dot u1 = 1
w2 = [1, 1, 0] - 1*[1, 0, 0] = [0, 1, 0]
u2 = [0, 1, 0]

For v3:

v3 dot u1 = 1, v3 dot u2 = 1
w3 = [1, 1, 1] - 1*[1, 0, 0] - 1*[0, 1, 0] = [0, 0, 1]
u3 = [0, 0, 1]

The vectors were already orthonormal!

Matrix Formulation

The Gram-Schmidt process can be expressed using matrix factorization: A = QR.

QR Decomposition

Given an m x n matrix A with linearly independent columns:

  • Q: m x n matrix with orthonormal columns (from Gram-Schmidt)
  • R: n x n upper triangular matrix

The entries of R are: R_ii = |w_i| and R_ij = v_i dot u_j for i > j.

Applications of Gram-Schmidt
Key Applications
  • Orthogonalization: Convert any basis to orthonormal
  • QR Factorization: Used in solving least squares problems
  • Computing Eigenvalues: The QR algorithm uses this repeatedly
  • Solving Linear Systems: When A = QR, solving Ax = b becomes easier
  • Numerical Stability: Numerical stability in algorithms
Properties
  • If the original vectors are linearly dependent, wk = 0 at some step
  • The span of {v1, ..., vk} = span of {u1, ..., uk} for all k
  • The process works in any finite-dimensional inner product space
  • Order of the original basis affects the resulting orthonormal basis
Numerical Considerations

The classical Gram-Schmidt process can suffer from numerical instability due to rounding errors. In practice, modified Gram-Schmidt or Householder transformations are preferred for computational applications.
