Chapter 4 Summary
eigenvalues, Linear Transformations, Vector Spaces

This chapter covers eigenvalues/eigenvectors, diagonalization, linear transformations, kernels, range, and orthogonality.

eigenvalues & eigenvectors
Definition
eigenvalue Equation
Fundamental
$$A\vec{v} = \lambda\vec{v}$$

Find eigenvalues: $$\det(A - \lambda I) = 0$$ (characteristic equation)

Example
Finding eigenvalues
Problem

$$A = \begin{bmatrix}2 & 1 \\ 1 & 2\end{bmatrix}$$

Characteristic eq: $$(2-\lambda)^2 - 1 = 0$$

$$(2-\lambda)^2 = 1 \Rightarrow 2-\lambda = \pm 1$$

eigenvalues: $$\lambda = 1$$ and $$\lambda = 3$$

Linear Transformations
Definition
Linear Transformation

$$T: \mathbb{R}^n \to \mathbb{R}^m$$ is linear if $$T(\vec{u}+\vec{v}) = T(\vec{u}) + T(\vec{v})$$ and $$T(c\vec{u}) = cT(\vec{u})$$

Definition
Kernel & Range

Kernel: All inputs mapping to zero

Range: All possible outputs

Orthogonality
Definition
Orthogonal Vectors
Condition
$$\vec{v} \cdot \vec{w} = 0$$
Definition
Unit Vector
Formula
$$\hat{u} = \frac{\vec{u}}{|\vec{u}|}$$
