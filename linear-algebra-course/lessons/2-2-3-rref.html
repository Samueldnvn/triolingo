Lesson 2.2.3: Reduced Row Echelon Form (RREF)

Module 2.2: Linear Systems | Chapter 2: Matrices & Systems

RREF

RREF (Reduced Row Echelon Form) is a canonical form where:

  • Leading entry in each nonzero row is 1
  • Each leading 1 is to the right of the one above
  • Leading 1 is the only nonzero entry in its column
  • Rows of zeros are at the bottom
RREF Example

[[2, 4|8], [3, 6|12]]

R1 -> R1/2: [[1, 2|4], [3, 6|12]]

R2 -> R2 - 3R1: [[1, 2|4], [0, 0|0]]

RREF: [[1, 2|4], [0, 0|0]]

Interpretation: x + 2y = 4. Infinite solutions: x = 4 - 2t, y = t (free variable)

Practice Problems
Problem 1
What is the RREF of [[2, 0], [0, 3]]?
Problem 2
Is [[1, 0, 0], [0, 1, 0], [0, 0, 1]] in RREF? (yes/no)
