Eigenvalue

A scalar lambda is an eigenvalue of matrix A if there exists a nonzero vector v (eigenvector) such that:

Av = lambda v
Characteristic Equation

To find eigenvalues, solve det(A - lambda I) = 0

Example

Find eigenvalues of [[2, 1], [1, 2]]

A - lambda I = [[2-lambda, 1], [1, 2-lambda]]

det = (2-lambda)^2 - 1 = 0

(2-lambda)^2 = 1, so 2-lambda = plus/minus 1

Eigenvalues: lambda = 1 and lambda = 3

Diagonal Matrix Eigenvalues

For diagonal matrix, eigenvalues are simply the diagonal entries!
