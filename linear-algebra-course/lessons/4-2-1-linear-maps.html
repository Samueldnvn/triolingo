Linear Transformation

A function T: R^n -> R^m is linear if:

1. T(x + y) = T(x) + T(y) (additivity)

2. T(kx) = kT(x) (homogeneity)

Example

T(x, y) = (2x + y, x - 3y)

This is linear because:

T(2, 3) = (7, -7)

T(2*(1, 1)) = T(2, 2) = (6, -4)

2*T(1, 1) = 2*(3, -2) = (6, -4)
