Gaussian Elimination

An algorithm for solving systems of linear equations using elementary row operations to transform the matrix into row echelon form (REF).

Elementary Row Operations
The Three Operations

1. Swap two rows (Ri <-> Rj)

2. Multiply a row by nonzero scalar (kRi -> Ri)

3. Add multiple of one row to another (Ri + kRj -> Ri)

Example

Solve using Gaussian elimination:

[[1, 2|3], [4, 5|6]]

Step 1: R2 -> R2 - 4R1

[[1, 2|3], [0, -3|-6]]

Step 2: R2 -> R2 / (-3)

[[1, 2|3], [0, 1|2]]

Step 3: R1 -> R1 - 2R2

[[1, 0|-1], [0, 1|2]]

Solution: x = -1, y = 2
