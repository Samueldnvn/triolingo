Diagonalization

A matrix A is diagonalizable if A = PDP^(-1) where D is diagonal and P contains eigenvectors.

Process

1. Find eigenvalues

2. Find independent eigenvectors

3. Form P from eigenvectors

4. Form D from eigenvalues
