Lesson 3.1.3: Determinant Properties

Key Properties
1. Multiplicative Property

det(AB) = det(A) * det(B)

2. Determinant of Transpose

det(A^T) = det(A)

3. Scalar Multiple

det(kA) = k^n * det(A) where n is matrix dimension

4. Determinant of Inverse

det(A^(-1)) = 1/det(A)

5. Triangular Matrix

For triangular matrix, det = product of diagonal entries

Example

If det(A) = 2 and det(B) = 3, find det(AB)

det(AB) = det(A) * det(B) = 2 * 3 = 6
