Linear System

A system of linear equations is a collection of one or more linear equations involving the same set of variables.

Example System
System Example

x + y = 3

x - y = 1

Adding: 2x = 4, so x = 2. Then y = 3 - x = 1. Solution: (2, 1)

Matrix Form

A linear system Ax = b can be written in augmented matrix form [A|b]

Example

System: x + y = 3, 2x + 3y = 7

Matrix form: [[1, 1|3], [2, 3|7]]
