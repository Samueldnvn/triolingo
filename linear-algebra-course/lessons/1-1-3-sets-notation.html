Introduction to Sets

A set is a fundamental concept in mathematics - a collection of distinct objects, called elements or members. Understanding sets is crucial for linear algebra, where we'll work with sets of vectors, sets of matrices, and mathematical spaces. The precision and clarity of set notation will serve you throughout your mathematical journey.

In this lesson, we'll learn how to describe sets, use mathematical notation, perform set operations, and understand important set relationships. These concepts form the language of advanced mathematics!

What is a Set?
Definition
Set

A set is a well-defined collection of distinct objects called elements. Sets are usually denoted by capital letters like $$A, B, C, \dots$$

Example
Describing Sets
Example 1

$$A = \{1, 2, 3, 4, 5\}$$ - the set containing the numbers 1, 2, 3, 4, and 5

Example 2

$$B = \{\text{apple}, \text{banana}, \text{orange}\}$$ - a set of fruits

Example 3

$$C = \{x : x \text{ is a positive integer}\}$$ - the set of all positive integers (described using set-builder notation)

Important
Key Facts About Sets
  • Elements are unique: A set doesn't contain duplicates
  • Order doesn't matter: $$\{1, 2, 3\} = \{3, 2, 1\}$$
  • Empty set: $$\emptyset$$ or $$\{\}$$ is a set with no elements
  • Set membership: $$x \in A$$ means "$$x$$ is an element of set $$A$$"
  • Not in set: $$x \notin A$$ means "$$x$$ is NOT an element of set $$A$$"
Set Notation
Notation 1
Roster Method

List all elements inside curly braces, separated by commas:

Example
$$A = \{2, 4, 6, 8, 10\}$$
Notation 2
Set-Builder Notation

Describe the property that all elements must satisfy:

Pattern
$$\{x : P(x)\}$$ or $$\{x \mid P(x)\}$$

Read as: "the set of all $$x$$ such that property $$P(x)$$ is true"

Example
Set-Builder Notation Examples
Example 1

$$\{x : x > 0\}$$ The set of all positive numbers

Example 2

$$\{n : n \text{ is an even integer}\}$$ The set of all even integers

Example 3

$$\{x^2 : x \in \mathbb{Z}\}$$ All perfect squares of integers

This equals: $$\{0, 1, 4, 9, 16, 25, \dots\}$$

Example 4

$$\{2n : n \in \mathbb{N}\}$$ All even natural numbers

This equals: $$\{2, 4, 6, 8, 10, \dots\}$$

Famous Number Sets
Standard Notation
  • $$\mathbb{N} = \{1, 2, 3, \dots\}$$ Natural numbers
  • $$\mathbb{W} = \{0, 1, 2, \dots\}$$ Whole numbers
  • $$\mathbb{Z} = \{\dots, -2, -1, 0, 1, 2, \dots\}$$ Integers
  • $$\mathbb{Q} = \{\frac{a}{b} : a, b \in \mathbb{Z}, b \neq 0\}$$ Rational numbers
  • $$\mathbb{R}$$ Real numbers
  • $$\mathbb{C}$$ Complex numbers
Set Relationships
Relationship 1
Subset

$$A \subseteq B$$ means "$$A$$ is a subset of $$B$$" - every element of $$A$$ is also in $$B$$

Formal
$$A \subseteq B \iff \forall x, x \in A \implies x \in B$$
Relationship 2
Proper Subset

$$A \subset B$$ means "$$A$$ is a proper subset of $$B$$" - $$A \subseteq B$$ but $$A \neq B$$

Relationship 3
Equal Sets

$$A = B$$ means "$$A$$ equals $$B$$" - both contain exactly the same elements

Formal
$$A = B \iff A \subseteq B \text{ AND } B \subseteq A$$
Relationship 4
Disjoint Sets

$$A$$ and $$B$$ are disjoint if they have no common elements

Formal
$$A \cap B = \emptyset$$
Example
Set Relationship Examples
Example 1

Let $$A = \{1, 2, 3\}$$ and $$B = \{1, 2, 3, 4, 5\}$$

$$A \subseteq B$$ ✓ (A is a subset of B)

$$A \subset B$$ ✓ (A is a PROPER subset of B)

Example 2

Let $$C = \{2, 3\}$$

$$C \subseteq A$$ ✓

$$C \subseteq B$$ ✓

$$\mathbb{N} \subseteq \mathbb{Z} \subseteq \mathbb{Q} \subseteq \mathbb{R}$$

Example 3: Disjoint

Let $$D = \{1, 3, 5\}$$ and $$E = \{2, 4, 6\}$$

$$D \cap E = \emptyset$$, so D and E are disjoint

Set Operations
Operation 1
Union

$$A \cup B = \{x : x \in A \text{ OR } x \in B\}$$ - all elements in A, B, or both

Example
$$\{1, 2\} \cup \{2, 3, 4\} = \{1, 2, 3, 4\}$$
Operation 2
Intersection

$$A \cap B = \{x : x \in A \text{ AND } x \in B\}$$ - elements common to both A and B

Example
$$\{1, 2, 3, 4\} \cap \{3, 4, 5, 6\} = \{3, 4\}$$
Operation 3
Difference

$$A \setminus B = \{x : x \in A \text{ AND } x \notin B\}$$ - elements in A but NOT in B

Example
$$\{1, 2, 3, 4, 5\} \setminus \{3, 4\} = \{1, 2, 5\}$$
Operation 4
Complement

$$A^c = U \setminus A$$ where $$U$$ is the universal set - all elements NOT in A

Example
If $$U = \{1, 2, 3, 4, 5\}$$ and $$A = \{2, 4\}$$, then $$A^c = \{1, 3, 5\}$$
Example
Combining Set Operations
Problem

Let $$A = \{1, 2, 3, 4, 5\}$$, $$B = \{3, 4, 5, 6, 7\}$$, $$C = \{5, 6, 7, 8, 9\}$$

a) $$A \cup B$$

$$\{1, 2, 3, 4, 5, 6, 7\}$$

b) $$A \cap B$$

$$\{3, 4, 5\}$$

c) $$(A \cup B) \cap C$$

First find $$A \cup B = \{1, 2, 3, 4, 5, 6, 7\}$$

Then find intersection with C: $$\{5, 6, 7\}$$

d) $$A \setminus C$$

$$\{1, 2, 3, 4\}$$

Important Properties
De Morgan's Laws for Sets
  • $$(A \cup B)^c = A^c \cap B^c$$ - Complement of union
  • $$(A \cap B)^c = A^c \cup B^c$$ - Complement of intersection

These laws show how complements interact with unions and intersections!

Set Cardinality
Definition
Cardinality

The cardinality of a set, denoted by $$|A|$$, is the number of elements in the set.

Example
Cardinality Examples
Example 1

$$|\{1, 2, 3, 4, 5\}| = 5$$

Example 2

$$|\{\text{a, b, c, d}\}| = 4$$

Example 3

$$|\emptyset| = 0$$ (empty set has 0 elements)

Example 4: Infinite Sets

$$|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}| = \infty$$ (countably infinite)

$$|\mathbb{R}| = \infty$$ (uncountably infinite, "larger" than integers!)

Formula
Cardinality of Union

For finite sets:

$$|A \cup B| = |A| + |B| - |A \cap B|$$
