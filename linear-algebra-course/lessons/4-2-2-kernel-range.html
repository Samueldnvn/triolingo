Kernel (Null Space)

Kernel(T) = {v in domain : T(v) = 0}

The set of all inputs that map to zero.

Range (Image)

Range(T) = {T(v) : v in domain}

The set of all possible outputs.

Inner Products

Inner product satisfies: positive definite, symmetric, linear

Dot product is a common inner product.

Projection

Projection of v onto unit vector u: proj_u(v) = (v . u)u

Gram-Schmidt Process

Converts linearly independent vectors into orthonormal basis.
