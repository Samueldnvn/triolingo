Singular Value Decomposition
Definition
SVD
Formula
$$A = U\Sigma V^T$$
Components
  • $$U$$: Orthogonal matrix (left singular vectors)
  • $$\Sigma$$: Diagonal matrix (singular values)
  • $$V^T$$: Orthogonal matrix (right singular vectors)
Applications
Uses
  • PCA: Principal component analysis for data reduction
  • Image Compression: SVD for reducing image complexity
  • Recommender Systems: Matrix factorization methods
  • Statistics: Covariance matrices, correlation
