Module 2.2 Summary
Linear Systems & Gaussian Elimination

This module covers solving systems of linear equations using matrices and row operations.

Definition
Linear System
A system can be written as $$Ax=b$$ where A is the coefficient matrix, x is unknowns, b is constants.
Method
Gaussian Elimination
Use row operations to transform to row-echelon form
Form
RREF
Reduced Row Echelon Form: leading 1s, zeros elsewhere in each pivot column
Example
Solving a System
Problem

$$2x+3y=7, x-y=1$$

Matrix form: $$\begin{bmatrix}2 & 3 \\ 1 & -1\end{bmatrix}\begin{bmatrix}x \\ y\end{bmatrix}=\begin{bmatrix}7 \\ 1\end{bmatrix}$$

Practice
Problem 1
Solve $$x+y=3, x-y=1$$ (Enter as "x=2,y=1")
