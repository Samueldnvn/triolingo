Vector Addition and Scalar Multiplication
Overview

Vectors can be added together and multiplied by scalars, similar to how we work with numbers but applied component-wise to all vector components simultaneously.

Definition
Vector Addition
Formula
$$\langle a, b \rangle + \langle c, d \rangle = \langle a+c, b+d \rangle$$
Definition
Scalar Multiplication
Formula
$$k\langle a, b \rangle = \langle ka, kb \rangle$$
Example
Vector Calculations
Example 1: Addition

$$\langle 1, 3 \rangle + \langle 2, 4 \rangle = \langle 3, 7 \rangle$$

Example 2: Scalar Multiplication

$$2\langle 3, 5 \rangle = \langle 6, 10 \rangle$$

Practice Problems
Problem 1
Calculate $$\langle 2, 1 \rangle + \langle 3, 4 \rangle$$
Problem 2
Calculate $$3\langle 1, 2 \rangle$$
Problem 3
