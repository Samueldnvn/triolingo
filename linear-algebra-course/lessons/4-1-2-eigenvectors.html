Eigenvector

A nonzero vector v is an eigenvector of matrix A with eigenvalue lambda if:

Av = lambda v
Or: (A - lambda I)v = 0
Example

Find eigenvector of [[2, 1], [1, 2]] for eigenvalue lambda = 3

A - 3I = [[-1, 1], [1, -1]]

[[-1, 1], [1, -1]] [x, y] = [0, 0]

-x + y = 0, so y = x

Eigenvectors: any nonzero multiple of [1, 1]
