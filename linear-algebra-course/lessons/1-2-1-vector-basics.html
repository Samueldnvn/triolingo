Introduction to Vectors

Vectors are the building blocks of linear algebra. While scalars are single numbers, vectors have both magnitude and direction, representing quantities in multi-dimensional space. They're used everywhere - physics (force, velocity), computer graphics (3D modeling), machine learning (data vectors), and more.

In this lesson, we'll learn what vectors are, how to represent them in coordinate form, and understand their geometric interpretation. Mastering vectors is essential for everything that follows in linear algebra!

What is a Vector?
Definition
Vector

A vector is an ordered collection of numbers called components. In $$\mathbb{R}^2$$ (2D), a vector has 2 components; in $$\mathbb{R}^3$$ (3D), it has 3 components; and more generally, in $$\mathbb{R}^n$$, it has n components.

Notation
$$\vec{v} = \langle v_1, v_2 \rangle$$ or $$\vec{v} = \begin{bmatrix} v_1 \\ v_2 \end{bmatrix}$$
Example
Vector Representations
Example 1

$$\vec{v} = \langle 3, 4 \rangle$$ - vector with components 3 and 4

Example 2

$$\vec{w} = \begin{bmatrix} -2 \\ 5 \end{bmatrix}$$ - same vector in column form

Example 3

$$\vec{u} = \langle 0, 0 \rangle = \vec{0}$$ - the zero vector

Geometric Interpretation
Geometry

In 2D, a vector $$\langle a, b \rangle$$ represents an arrow starting at the origin $$(0, 0)$$ and ending at the point $$(a, b)$$. The length of the arrow is the magnitude, and the direction points from the origin to $$(a, b)$$.

Example
Geometric Vectors
Example 1

$$\langle 1, 0 \rangle$$ - points right along x-axis

Example 2

$$\langle 0, 1 \rangle$$ - points up along y-axis

Example 3

$$\langle 2, 2 \rangle$$ - points diagonally, 45 degrees
