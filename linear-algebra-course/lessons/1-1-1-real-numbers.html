Introduction to Real Numbers

Real numbers are the foundation of mathematics. They include all the numbers we can find on the number line - from negative infinity to positive infinity, including all the fractions and decimals in between. Understanding real numbers is absolutely essential for linear algebra, as vectors and matrices are built entirely from real numbers.

In this lesson, we'll explore the different types of real numbers, their properties, absolute value, and how they behave in mathematical operations. By the end, you'll have a solid foundation for working with more complex mathematical structures.

Types of Real Numbers
Definition
Real Numbers

The set of all real numbers, denoted by $$\mathbb{R}$$, includes all numbers that can be represented on the number line. This includes:

  • Natural numbers: $$\mathbb{N} = \{1, 2, 3, 4, ...\}$$ - Counting numbers starting from 1
  • Whole numbers: $$\mathbb{W} = \{0, 1, 2, 3, ...\}$$ - Natural numbers plus zero
  • Integers: $$\mathbb{Z} = \{..., -3, -2, -1, 0, 1, 2, 3, ...\}$$ - All whole numbers and their negatives
  • Rational numbers: $$\mathbb{Q}$$ - Numbers that can be written as fractions $$\frac{a}{b}$$ where $$a, b \in \mathbb{Z}$$ and $$b \neq 0$$
  • Irrational numbers: Numbers that cannot be written as fractions, like $$\pi$$, $$e$$, and $$\sqrt{2}$$
Example
Classifying Numbers

Let's classify $$3, \frac{5}{2}, -7, 0, \pi$$:

  • $$3$$ is a natural number, whole number, integer, and rational number
  • $$\frac{5}{2} = 2.5$$ is a rational number (not an integer)
  • $$-7$$ is an integer and rational number
  • $$0$$ is a whole number, integer, and rational number
  • $$\pi \approx 3.14159...$$ is an irrational number (the decimal never repeats)
Absolute Value
Definition
Absolute Value

The absolute value of a real number $$x$$, denoted by $$|x|$$, is the distance from $$x$$ to 0 on the number line. It's always non-negative.

Formula
$$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$
Example
Calculating Absolute Values

Let's compute several absolute values step by step:

Example 1

$$|7| = 7$$ (already non-negative)

Example 2

$$|-7| = -(-7) = 7$$ (negative becomes positive)

Example 3

$$|0| = 0$$ (zero is a special case)

Example 4

$$|\frac{-5}{2}| = \frac{5}{2} = 2.5$$ (fractions work the same)

Example 5

$$|-\pi| = \pi \approx 3.14159...$$ (irrational numbers too!)

Properties of Real Numbers
Theorem
Absolute Value Properties

For any real numbers $$a, b$$:

  • Non-negativity: $$|a| \geq 0$$ and $$|a| = 0$$ only when $$a = 0$$
  • Multiplicativity: $$|ab| = |a| \cdot |b|$$
  • Triangle inequality: $$|a + b| \leq |a| + |b|$$
  • Reverse triangle inequality: $$|a - b| \geq ||a| - |b||$$
Example
Applying Properties
Example 1: Multiplicativity

Find $$|3 \cdot (-5)|$$:

Method 1: $$|3 \cdot (-5)| = |-15| = 15$$

Method 2: $$|3| \cdot |-5| = 3 \cdot 5 = 15$$

Both methods give us 15 ✓

Example 2: Triangle Inequality

Verify that $$|3 + (-4)| \leq |3| + |-4|$$

$$|3 + (-4)| = |-1| = 1$$

$$|3| + |-4| = 3 + 4 = 7$$

Since $$1 \leq 7$$, the inequality holds! ✓

Important
Key Insight

The triangle inequality is fundamental in linear algebra! It states that the distance directly from point $$a$$ to point $$b$$ is always less than or equal to going from $$a$$ to 0, then from 0 to $$b$$. This intuitively makes sense - the straight path is always shorter or equal to any detour.

Ordering and Comparison
Definition
Ordering Properties

Real numbers have a natural order. We compare using symbols:

  • $$a < b$$ means $$a$$ is less than $$b$$
  • $$a > b$$ means $$a$$ is greater than $$b$$
  • $$a \leq b$$ means $$a$$ is less than or equal to $$b$$
  • $$a \geq b$$ means $$a$$ is greater than or equal to $$b$$
Example
Comparing Numbers and Using Absolute Value
Example 1

Find the distance between $$3$$ and $$-7$$ on the number line:

This is $$|3 - (-7)| = |3 + 7| = |10| = 10$$

The distance is 10 units.

Example 2

Which is closer to $$0$$: $$\frac{7}{3}$$ or $$-\frac{5}{2}$$?

$$|\frac{7}{3}| = \frac{7}{3} \approx 2.33...$$

$$|-\frac{5}{2}| = \frac{5}{2} = 2.5$$

Since $$2.33... < 2.5$$, $$\frac{7}{3}$$ is closer to 0.

Real Numbers in Linear Algebra
Foundation
Why This Matters

Real numbers are the building blocks of everything in linear algebra:

  • Vectors: Each vector is a collection of real numbers
  • Matrices: Each matrix entry is a real number
  • Dot products: Compute real number results from vector operations
  • Determinants: Give real numbers as matrix properties
  • Eigenvalues: Real number special values of matrices

Mastering real number operations now will make everything else much easier!

Caution
Warning

Remember that while we'll focus on real numbers in this course, advanced linear algebra also uses complex numbers. The properties and operations you learn here extend to complex numbers as well!
