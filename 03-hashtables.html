When to Use Hash Tables

Pattern Recognition

Use hash tables when you need to:

  • Find if element exists (lookup)
  • Count frequencies
  • Map values to other values
  • Track seen elements
  • Find complements (target - current)

Two Sum

For each element, if (target - element) in hash, we found pair

Group Anagrams

Group by sorted string as key (all anagrams have same sorted string)

Common Hash Table Problems

Subarray Sum Equals K

Track prefix sums. If (current_sum - k) was seen before, subarray exists between those points

Longest Substring Without Repeating

Track last seen index of each character. Move left past last occurrence when duplicate found

Frequency Counting

Use dict: d[x] = d.get(x, 0) + 1 or collections.Counter
