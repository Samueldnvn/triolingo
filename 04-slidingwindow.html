Fixed-size Sliding Window

Maximum Sum of Subarray of Size K

Calculate first k elements sum. Then: add new, subtract leaving element. Track max.

def max_sum(arr, k):
    window_sum = sum(arr[:k])
    max_sum = window_sum
    for i in range(k, len(arr)):
        window_sum += arr[i] - arr[i-k]
        max_sum = max(max_sum, window_sum)
    return max_sum

Variable-size Sliding Window

Window expands when valid, shrinks when invalid

Key: Find condition to check validity. Expand right. When invalid, shrink left until valid.

Longest Substring with at most K Distinct

Use frequency map. Expand right. When distinct > k, shrink left. Track max length.

Minimum Window Substring Containing Pattern

Expand right until window contains all pattern chars. Then shrink left to find minimum.
