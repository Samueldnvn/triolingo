# IKEA Theme for Triolingo

A blue and yellow theme inspired by IKEA's iconic Swedish design aesthetic, now with official IKEA assets!

## Assets

### Background: `ikea-background-enhanced.svg`
- IKEA blue gradient background (#0058a3 to #003d6f)
- Grid pattern overlay
- Diagonal yellow stripe areas
- Furniture silhouettes in background (chair, table, bookshelf, lamp, cabinet, sofa)
- Large decorative yellow circles
- Official IKEA logo in corner
- IKEA's signature blue and yellow color scheme

### Official Logo: `ikea-logo-official.svg`
- **Downloaded from Wikimedia Commons**
- Official IKEA logo with oval background
- Authentic IKEA branding
- Blue oval with yellow IKEA text

### Custom Icons:
- `ikea-chair-icon.svg` - Simple chair icon in IKEA style
- `ikea-lamp-icon.svg` - Simple lamp icon in IKEA style

### Legacy Files:
- `ikea-background.svg` - Original background (replaced by enhanced version)
- `ikea-logo.svg` - Custom logo (replaced by official version)

## Color Scheme

- **Primary Blue**: #0058a3 (IKEA Blue)
- **Secondary Blue**: #004a8a
- **Hover Blue**: #003d6f
- **Accent Yellow**: #ffda00 (IKEA Yellow)
- **Hover Yellow**: #e5c800
- **Text**: #ffffff (White)
- **Borders**: #ffda00 (Yellow)

## Design Features

- Bold 3px yellow borders on all UI elements
- Blue backgrounds with yellow accents
- Yellow buttons with blue text
- Official IKEA logo integration
- Furniture silhouettes in background
- Clean, functional Swedish design aesthetic
- High contrast for excellent readability

## Usage

The IKEA theme is available in Triolingo's Settings → Themes menu. Simply select "IKEA" to apply the blue and yellow design theme with official IKEA branding!
