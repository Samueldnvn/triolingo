# Triolingo - Quick Start

## Run the app:

```bash
cd /home/samueldnvn4/.openclaw/workspace/triolingo
npm start
```

## Current features:
- Clean white interface
- French course with 8 lessons across 3 units
- Local progress saving (to app data folder)
- XP system
- Lesson tracking
- Clean language switching (extensible for other skills)

## Tech stack:
- Electron (desktop app framework)
- Vanilla JavaScript (no frameworks)
- CSS variables for easy theming

## To add more courses/skills:
Edit `app.js` and add to the `courses` object following the same structure.
