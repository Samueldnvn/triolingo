# GreenSheet - Complete Excel-like Desktop Spreadsheet

Created: 2026-02-21

## What It Does

A fully functional spreadsheet application built with Electron that mimics Microsoft Excel with a cohesive green theme matching the linear algebra course and memory palace app aesthetics.

## Key Features

### Spreadsheet Engine
- 1000 rows × 100 columns (grid: A1 to CV1000)
- Cell selection (single and multi-cell ranges with Shift+Click)
- Cell editing (double-click or press Enter)
- Cell addresses using Excel-style notation (A1, B5, Z100)

### Formula Engine (Custom Built)
30+ Excel-compatible functions:
- **Math**: SUM, AVERAGE, COUNT, MAX, MIN, ROUND, ABS, SQRT, POWER
- **Text**: CONCATENATE, LEFT, RIGHT, LEN, UPPER, LOWER, TRIM
- **Logical**: IF, AND, OR, NOT
- **Date/Time**: TODAY, NOW, DAY, MONTH, YEAR

Supports:
- Cell references (=A1+B1)
- Ranges (=SUM(A1:C100))
- Comparison operators (=, <>, <, >, <=, >=)
- Nested formulas

### Cell Formatting
- Text alignment (left, center, right)
- Vertical alignment (top, middle, bottom)
- Font styles (bold, italic, underline)
- Text color (color picker)
- Background color (color picker)
- Number formats (general, number, currency, percentage, date, time, text)
- Font family (5 options)
- Font size (10-24pt)

### Charts (5 Types)
- Bar charts
- Line charts
- Pie charts
- Doughnut charts
- Scatter charts

All charts automatically use green-themed colors matching the app aesthetic.

### File Operations
- New workbook (Ctrl+N)
- Open Excel/CSV files (Ctrl+O)
- Save workbook (Ctrl+S)
- Save As (.xlsx or .csv)
- Reads and writes actual Excel files using SheetJS

### Data Operations
- Copy (Ctrl+C)
- Paste (Ctrl+V)
- Cut (Ctrl+X)
- Delete (Delete key)
- Undo (Ctrl+Z) - 100-step history
- Redo (Ctrl+Y)
- Fill Down (Ctrl+D) - Copy formula down
- Fill Right (Ctrl+R) - Copy formula right
- Insert Row Above/Before
- Insert Column Before/After

### Sheet Management
- 3 sheets by default
- Add unlimited sheets via + button
- Rename sheets (double-click tab)
- Switch between sheets
- Navigation prev/next buttons

### User Interface
- Full-featured toolbar with icons
- Formula bar with cell address display
- Row headers (click to select row)
- Column headers (click to select column)
- Sheet tabs at bottom
- Status bar showing mode, selection, cell count
- Format cells dialog
- Insert chart dialog
- Formulas reference dialog

### Keyboard Shortcuts
- Navigation: Arrow keys, Enter
- Editing: Enter, Escape, Tab
- Formatting: Ctrl+B (bold), Ctrl+I (italic), Ctrl+U (underline)
- Operations: Ctrl+C/V/X/Z/Y (copy/paste/cut/undo/redo)
- File: Ctrl+N/O/S (new/open/save)
- Fill: Ctrl+D (down), Ctrl+R (right)
- Calculate: F9

## Implementation Notes

The formula engine is fully custom, parsing and evaluating formulas inline without external dependencies. This allows:
- Cell reference resolution (A1 → row 0, col 0)
- Range expansion (A1:C5 → [A1, B1, C1, A2, ...])
- Function execution (SUM, AVERAGE, etc.)
- Error handling (#ERROR)

State management uses:
- In-memory workbook object
- History array for undo/redo
- Selection tracking (start/end points)
- Active cell reference
- Modified file flag

Chart.js is loaded via CDN for visualization flexibility.
