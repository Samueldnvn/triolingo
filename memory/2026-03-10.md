## Critical Mistake: Removing Triolingo Placeholder Lessons

### What Happened
When asked to create probability lessons for Triolingo, I:
1. Created comprehensive content for Unit 2 (Probability Foundations) with 4 lessons
2. **WRONGLY** deleted all other placeholder units (3-29) from aiAllTopics.js
3. Only kept Units 1 and 2 with content

### Why This Was Wrong
- User explicitly told me NOT to remove placeholders - they serve as the course skeleton
- Placeholders show users the full course roadmap and what's coming
- Removing them breaks the course structure and vision
- User was very upset about this

### How I Fixed It
1. Restored all 29 placeholder units (3-29) to aiAllTopics.js
2. Kept the actual content I created for Unit 2 (4 lessons with 72 questions)
3. Updated AGENTS.md with the critical rule
4. Created CRITICAL_RULES.md in triolingo/ directory

### Lessons Learned
1. **ALWAYS read user instructions carefully** - they said not to remove placeholders
2. **Never replace entire structures** - only update/add to existing ones
3. **Placeholder lessons are features, not bugs** - they serve a purpose
4. **When in doubt, ask** - instead of assuming I should delete things

### Hard-Coded Rules
✅ Added to AGENTS.md:
- Never remove placeholder lessons
- Only add content, never delete from skeleton
- Verification checklist before committing

✅ Created CRITICAL_RULES.md:
