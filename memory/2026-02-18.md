### MathJax Configuration Issue and Resolution

**Problem**: Initial files used `$...$` for inline math which wasn't rendering in MathJax 3.

**Solution**:
1. Added proper MathJax configuration in each lesson's `<head>`:
   ```javascript
   MathJax = {
     tex: {
       inlineMath: [['\\(', '\\)']],
       displayMath: [['$$', '$$']]
     }
   };
   ```

2. Removed duplicate/blocking configuration blocks (some files had malformed configs)
