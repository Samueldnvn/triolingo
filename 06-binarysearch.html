Standard Binary Search

Algorithm

while left <= right: mid = left + (right-left)//2. Check arr[mid] vs target. Adjust left/right.

Key: Use left + (right-left)//2 to avoid overflow.

First/Last Occurrence

Don't return immediately on found. Keep searching half for first occurrence (move right), last (move left).

Rotated Array

One half is always sorted. Check if target in sorted half, search there. Else search other half.

Binary Search on Answer

Search Decision Space, Not Array

Find minimal/maximal value satisfying predicate. Define range [low, high]. Test mid with predicate function.

Examples

  • Minimum capacity ship packages
  • Split array largest sum
  • Minimum time to eat bananas
