Advanced Data Structures

Segment Tree

Range queries + point updates in O(log n). O(4n) space. Use for range min/max/sum queries with updates.

Fenwick Tree (Binary Indexed Tree)

Prefix sum queries + point updates in O(log n). Simpler than segment tree, often faster.

Trie

Prefix tree for strings. O(L) operations where L = word length. Used for prefix matching, autocomplete.

String Algorithms

KMP (Knuth-Morris-Pratt)

Pattern matching. Preprocess LPS array. O(n + m) time. No backtracking.

Rabin-Karp

Pattern matching using hash. Rolling hash. Good for multiple patterns or finding all occurrences.

Rolling Hash

Efficiently compute hash of substring. Remove old character contribution, add new. O(1) update.

Backtracking

Technique

Explore all possibilities recursively. Prune branches that can't lead to solution. Used for combinatorial problems.

Applications

N-Queens, Sudoku, generating permutations/subsets, combination sum problems.

Advanced Problem Solving

Bitmask DP

State includes subset as bitmask (bits = elements included/selected). Used for TSP, assignment, subset DP.

Monotonic Stack

Maintain order. Used for next greater/lesser elements, daily temperatures, histogram problems.

Meet in the Middle

For n ~ 30-50 when 2^n too large. Split in half, generate 2^(n/2) possibilities, combine results.
